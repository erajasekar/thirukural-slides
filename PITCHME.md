### பொறாமை குணத்தை எப்படி நீக்குறது?

---?include=includes/Alukkaramai.md

### அதை தெரிந்து கொள்ள இந்த காணொளியை இறுதி வரை பாருங்க. 

---?include=includes/Alukkaramai.md

### பொறாமை இல்லாமல் வாழ்வது மிகவும் முக்கியம்.

---?include=includes/Alukkaramai.md

### நம் வாழ்வில் மிக முக்கியமாக கடைபிடிக்க வேண்டிய ஒழுக்கம் எது ?

---?include=includes/Alukkaramai.md

```
ஒழுக்காறாக் கொள்க ஒருவன்தன் நெஞ்சத்து 
அழுக்காறு இலாத இயல்பு		( குறள் - 161)

```

@snap[span-100 text-08 text-left]

> உள்ளத்துள் பொறாமை இல்லாமல் வாழும் குணத்தை, ஒருவன் தனக்கு உரிய ஒழுக்கமாகக் கொள்க.

@snapend

---?include=includes/Alukkaramai.md

### ஏன் பொறாமை குணம் இல்லாமல் வாழ வேண்டும் ?

---?include=includes/Alukkaramai.md

```
விழுப்பேற்றின் அஃது ஒப்பது இல்லை யார் மாட்டும்
அழுக்காற்றின் அன்மை பெறின்		( குறள் - 162)

```

@snap[span-100 text-08 text-left]

> யாரிடமும் பொறாமை கொள்ளாத பண்பு ஒருவர்க்கு வாய்க்கப் பெறுமேயானால் அதற்கு மேலான பேறு அவருக்கு வேறு ஏதும் இல்லை.

@snapend

---?include=includes/Alukkaramai.md

### பொறாமை குணம் இருப்பவர்கள் எப்படி பட்டவர்கள்?

---?include=includes/Alukkaramai.md

```
அறன் ஆக்கம் வேண்டாதான் என்பான் பிறனாக்கம்
பேணாது அழுக்கறுப் பான்		( குறள் - 163)

```

@snap[span-100 text-08 text-left]

> தனக்கு அறமும் ஆக்கமும் விரும்பாதவன் என்று கருதத் தக்கவனே, பிறனுடைய ஆக்கத்தைக் கண்டு மகிழாமல் அதற்காகப் பொறாமைப்படுவான்.

@snapend

---?include=includes/Alukkaramai.md

### பொறாமை குணம் இருந்தால் என்ன ஆகும் ?

---?include=includes/Alukkaramai.md

```
அழுக்காற்றின் அல்லவை செய்யார் இழுக்காற்றின்
ஏதம் படுபாக்கு அறிந்து		( குறள் - 164)

```

@snap[span-100 text-08 text-left]

> தீய வழியில் சென்றால் துன்பம் ஏற்படுமென்பதை அறிந்தவர்கள் பொறாமையினால் தீச்செயல்களில் ஈடுபடமாட்டார்கள்.

@snapend

---?include=includes/Alukkaramai.md

### பொறாமை குணம் இருந்தால் துன்பம் யாரால் வரும்?

---?include=includes/Alukkaramai.md

```
அழுக்காறு உடையார்க்கு அதுசாலும் ஒன்னார்
வழுக்கியும்  கேடுஈன் பது		( குறள் - 165)

```

@snap[span-100 text-08 text-left]

> பொறாமை உடையவருக்குத் தீமை தரப் பகைவர் வேண்டியதில்லை. பொறாமையே போதும்.

@snapend

---?include=includes/Alukkaramai.md

### எதற்காக பொறாமைப் படுவது மிக கீழ்தரமானது?

---?include=includes/Alukkaramai.md

```
கொடுப்பது அழுக்கறுப்பான் சுற்றம் உடுப்பதூஉம்
உண்பதூஉம் இன்றிக் கெடும்		( குறள் - 166)

```

@snap[span-100 text-08 text-left]

> உதவியாக ஒருவருக்குக் கொடுக்கப்படுவதைப் பார்த்துப் பொறாமை கொண்டால் அந்தத் தீய குணம், அவனை மட்டுமின்றி அவனைச் சார்த்திருப்போரையும் உணவுக்கும், உடைக்கும்கூட வழியில்லாமல் ஆக்கிவிடும்.

@snapend

---?include=includes/Alukkaramai.md

### நிறைய நல்ல குணங்கள்  உள்ள நான் சில சமயங்களில் அடுத்தவரை பார்த்து பொறாமைப் பட்டால் என்ன ஆகும்?

---?include=includes/Alukkaramai.md

```
அவ்வித்து அழுக்காறு உடையானைச் செய்யவள்
தவ்வையைக் காட்டி விடும்		( குறள் - 167)

```

@snap[span-100 text-08 text-left]

> பிறர் உயர்வு கண்டு பொறாமைப்படுபவனைப் பார்க்கும் திருமகள் வெறுப்புக் கொண்டு மூதேவிக்கு அவனை அடையாளம் காட்டிவிட்டு விலகிப் போய்விடுவாள்.

@snapend

---?include=includes/Alukkaramai.md

### பொறாமை குணம் அவ்வளவு மோசமானதா?

---?include=includes/Alukkaramai.md

```
அழுக்காறு எனஒரு பாவி திருச்செற்றுத்
தீயுழி உய்த்து விடும்		( குறள் - 168)

```

@snap[span-100 text-08 text-left]

> பொறாமை எனப்படும் ஒப்பில்லாத பாவி எவனிடம் இருக்கிறதோ, அவனது செல்வத்தை அழிப்பதோடு, அவனை நரகத்திலும் அது சேர்க்கும்.

@snapend

---?include=includes/Alukkaramai.md

### பொறாமைப் படுபவருள் சிலர் நன்றாக வாழ்கிறார்களே ?

---?include=includes/Alukkaramai.md

```
அவ்விய நெஞ்சத்தான் ஆக்கமும் செவ்வியான்
கேடும் நினைக்கப் படும்		( குறள் - 169)

```

@snap[span-100 text-08 text-left]

> பொறாமை கொண்டவரின் உயர்வும் , அது இல்லாத நல்லவரின் தாழ்வும் ஆராயப் பட  வேண்டியது.

@snapend

---?include=includes/Alukkaramai.md

### பொறாமை குணம் இருந்தால் என்ன?

---?include=includes/Alukkaramai.md

```
அழுக்கற்று அகன்றாரும் இல்லை அஃது இல்லார்
பெருக்கத்தில் தீர்ந்தாரும் இல்		( குறள் - 170)

```

@snap[span-100 text-08 text-left]

> பொறாமை கொண்டு உயர்ந்தவரும் இல்லை. அது இல்லாத போது தாழ்ந்தவரும் இல்லை.

@snapend
