### புறங்கூறாமல் இருப்பது எவ்வளவு நல்லது ?

---?include=includes/Purankooramai.md

```
அறங்கூறா னல்ல செயினும் ஒருவன்
புறங்கூறா னென்றல் இனிது		( குறள் - 181)

```

@snap[span-100 text-08 text-left]

> ஒருவன் அறத்தைச் சொல்லாமல் பாவமே செய்தாலும், அடுத்தவரைப் பற்றிப் புறம் பேசமாட்டான் என்றால் அதுவே அவனுக்கு நல்லது.

@snapend

---?include=includes/Purankooramai.md

### புறங்கூறுதல் எவ்வளவு தீமையானது?

---?include=includes/Purankooramai.md

```
அறனழீஇ யல்லவை செய்தலின் தீதே
புறனழீஇப் பொய்த்து நகை		( குறள் - 182)

```

@snap[span-100 text-08 text-left]

> அறம் என்பதே இல்லை என்று அடித்துப் பேசிப் பாவத்தைச் செய்வதைக் காட்டிலும் ஒருவனைக் காணாதபோது புறம்பேசிக் காணும்போது பொய்யாகச் சிரிப்பது பெருங்கேடு.

@snapend

---?include=includes/Purankooramai.md

### புறம்சொல்லி வாழ்வது எவ்வளவு தாழ்வானது ?

---?include=includes/Purankooramai.md

```
புறங்கூறிப் பொய்த்துயிர் வாழ்தலிற் சாதல்
அறங்கூறும் ஆக்கந் தரும்		( குறள் - 183)

```

@snap[span-100 text-08 text-left]

> காணாதபோது ஒருவரைப் பற்றிப் புறம்பேசிக், காணும்போது பொய்யாக அவனுடன் பேசி வாழ்வதைக் காட்டிலும் இறந்து போவது அறநூல்கள் கூறும் உயர்வைத் தரும்.

@snapend

### ஒருவர் விரும்பதகாத செயல்கள் செய்பவரை குறை சொல்வது இயல்புத்தானே?

---?include=includes/Purankooramai.md

```
கண்ணின்று கண்ணறச் சொல்லினுஞ் சொல்லற்க
முன்னின்று பின்னோக்காச் சொல்		( குறள் - 184)

```

@snap[span-100 text-08 text-left]

> எதிரே நின்று கருணை இல்லாமல் கடுமையாகச் சொன்னாலும் சொல்லலாம், நேரில் இல்லாதபோது  பின் விளைவை ஆராயாத சொல்லைச் சொல்லக்கூடாது.

@snapend

---?include=includes/Purankooramai.md

### ஒருவர் நல்ல பண்புகளுடன் வாழ்கின்றாரா என்பதை எப்படி தெரிந்துகொள்வது ?

---?include=includes/Purankooramai.md

```
அறஞ்சொல்லும் நெஞ்சத்தான் அன்மை புறஞ்சொல்லும்
புன்மையாற் காணப் படும்		( குறள் - 185)

```

@snap[span-100 text-08 text-left]

> அறத்தைப் பெரிதாகப் பேசும் ஒருவன் மனத்தால் அறவோன் அல்லன் என்பதை அவன் புறம்பேசும் இழிவினைக் கொண்டு கண்டுகொள்ளலாம்.

@snapend

---?include=includes/Purankooramai.md

### அடுத்தவரை பற்றி புறம் பேசினால் என்ன?

---?include=includes/Purankooramai.md

```
பிறன்பழி கூறுவான் தன்பழி யுள்ளுந்
திறன்தெரிந்து கூறப் படும்		( குறள் - 186)

```

@snap[span-100 text-08 text-left]

> பிறர்மீது ஒருவன் புறங்கூறித் திரிகிறான் என்றால் அவனது பழிச் செயல்களை ஆராய்ந்து அவற்றில் கொடுமையானவைகளை அவன் மீது கூற நேரிடும்.

@snapend

---?include=includes/Purankooramai.md

### புறம் பேசினால் நட்பு வளரும் தானே ?

---?include=includes/Purankooramai.md

```
பகச்சொல்லிக் கேளிர்ப் பிரிப்பர் நகச்சொல்லி
நட்பாடல் தேற்றா தவர்		( குறள் - 187)

```

@snap[span-100 text-08 text-left]

> கூடி மகிழுமாறு இனியன பேசி நட்பை வளர்க்க தெரியாதவர்கள், நட்புக் கெடுமளவுக்குப் புறங்கூறி நண்பர்களை இழந்து விடுவார்கள்.

@snapend

---?include=includes/Purankooramai.md

### புறம் பேசுபவர்களுடன் நாம் நண்பர்களாக இருக்க முடியுமா?

---?include=includes/Purankooramai.md

```
துன்னியார் குற்றமுந் தூற்றும் மரபினர்
என்னைகொல் ஏதிலார் மாட்டு		( குறள் - 188)

```

@snap[span-100 text-08 text-left]

> நெருங்கிப் பழகியவரின் குறையைக்கூட புறம் பேசித் தூற்றுகின்ற குணமுடையவர்கள் அப்படிப் பழகாத அயலாரைப் பற்றி என்னதான் பேச மாட்டார்கள்!

@snapend

---?include=includes/Purankooramai.md

### புறம்பேசுபவர்களை பொறுத்து போகமுடியுமா?

---?include=includes/Purankooramai.md

```
அறனோக்கி யாற்றுங்கொல் வையம் புறனோக்கிப்
புன்சொ லுரைப்பான் பொறை		( குறள் - 189)

```

@snap[span-100 text-08 text-left]

> ஒருவர் நேரில் இல்லாத போது பழிச்சொல் கூறுவோனுடைய உடலை இவனைச் சுமப்பதும் அறமே என்று கருதித்தான் நிலம் சுமக்கிறது.

@snapend

---?include=includes/Purankooramai.md

### புறங்கூறுதலை நீக்க வழி என்ன?

---?include=includes/Purankooramai.md

```
ஏதிலார் குற்றம்போல் தங்குற்றங் காண்கிற்பின்
தீதுண்டோ மன்னு முயிர்க்கு		( குறள் - 190)

```

@snap[span-100 text-08 text-left]

> பிறர் குற்றத்தைக் காண்பவர்கள் தமது குற்றத்தையும் எண்ணிப் பார்ப்பார்களேயானால் புறங்கூறும் பழக்கமும் போகும். வாழ்க்கையும் நிம்மதியாக அமையும்.

@snapend
