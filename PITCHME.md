## திருக்குறள் அறிமுகம்

@img[width=300, height=350 , shadow](assets/img/valluvar.png)

2000 ஆண்டுகளுக்கு மேல் பழமையான இந்த நூலை இயற்றியவர் **திருவள்ளுவர்.**

---

### திருக்குறளின் உண்மையான பெயர் முப்பால். 
 
அறத்தை வலியுறுத்துவதால், இந்நூல் **அற நூல்** என்று சொல்லப் படுகிறது.

---
### இதன் வேறு பெயர்கள்

* உலகப்  பொது மறை
* பொய்யா மொழி
* உத்திர வேதம்
* தெய்வ  நூல் 
* வான் மறை 
* ஈரடி நூல் 
* தமிழ் மறை 

---

### திருக்குறளின் பயன் என்ன?

அறத்தை தெரிந்து கொண்டு அதன் படி வாழ்வது.

---

### அறம் என்றால் என்ன?

```
மனத்துக்கண் மாசிலன் ஆதல் அனைத்தறன்
ஆகுல நீர பிற. 
```

> மனத்தூய்மையே **அறம்**.

---

### மனத்தூய்மை என்றால் என்ன?

```
அழுக்காறு அவாவெகுளி இன்னாச்சொல் நான்கும்
இழுக்கா இயன்றது அறம்.
```

> பொறாமை, பேராசை, கோபம், புண்படுத்தும் சொல் ஆகியவை
> இல்லாமல் வாழ்வதே **அறம்** .

---

### ஏன் அற வழியில் வாழ வேண்டும் ?

```
அறத்தான் வருவதே இன்பம் மற்றெல்லாம்
புறத்த புகழும் இல.
```

> நாம் அற வழியில்  ஈட்டுகின்ற பொருளும் புகழுமே  நமக்கு  இன்பம்  தரும்.
---

### திருக்குறள் வரிசை படுத்தப்பட்டுள்ள முறை 

@img[width=450, height=450 , shadow](assets/img/thirukural-organization1.png)

---?include=includes/KadavulVazhthu.md

```
அகர முதல எழுத்தெல்லாம் ஆதி
பகவன் முதற்றே உலகு.            ( குறள் - 1 )
```

@snap[span-100 text-gray text-left text-04]
**தொடரமைப்பு:** எழுத்தெல்லாம் அகர முதல, உலகம் ஆதி பகவன் முதற்றே
@snapend

@snap[span-100 text-06 text-left]
> அகரம் எழுத்துக்களுக்கு முதன்மை; ஆதிபகவன், உலகில் வாழும் உயிர்களுக்கு முதன்மை.
@snapend

---?include=includes/KadavulVazhthu.md

```
கற்றதனால் ஆய பயனென்கொல் வாலறிவன்
நற்றாள் தொழாஅர் எனின்.        ( குறள் - 2 )
```

@snap[span-100 text-gray text-left text-04]
**தொடரமைப்பு:** வாலறிவன் நற்றாள் தொழாஅர் எனின், கற்றதனால் ஆய பயன் என்கொல்?
@snapend

@snap[span-100 text-06 text-left]
> இறைவனுடைய திருவடியை வணங்கும் பண்பு இல்லாவிடில் என்னதான் ஒருவர் கற்றிருந்தாலும் அதனால் என்ன பயன்? ஒன்றுமில்லை.
@snapend

---?include=includes/KadavulVazhthu.md

```
மலர்மிசை ஏகினான் மாணடி சேர்ந்தார்
நிலமிசை நீடுவாழ் வார்.         ( குறள் - 3 )
```

@snap[span-100 text-06 text-left]
> அன்பர்களின் உள்ளமாகிய மலரில் வாழ்பவனாகிய இறைவனின் திருவடியை மனதிலே இடைவிடாது நினைத்து வாழ்பவரின் புகழ் வாழ்வு , உலகில் நெடுங்காலம் நிலைத்து நிற்கும்.
@snapend

---?include=includes/KadavulVazhthu.md

```
வேண்டுதல் வேண்டாமை இலானடி சேர்ந்தார்க்கு
யாண்டும் இடும்பை இல.         ( குறள் - 4 )
```

@snap[span-100 text-06 text-left]
> விருப்பு வெறுப்பற்ற இறைவனையே என்றும் மனதில் நினைப்பவர்களுக்கு அத்தன்மையே ஏற்படுவதால்  அவர்களுக்கு எப்போதும் துன்பம் இல்லை.
@snapend

---?include=includes/KadavulVazhthu.md

```
இருள்சேர் இருவினையும் சேரா இறைவன்
பொருள்சேர் புகழ்புரிந்தார் மாட்டு.        ( குறள் - 5 )
```

@snap[span-100 text-gray text-left text-04]
**தொடரமைப்பு:** இறைவன் பொருள் சேர் புகழ் புரிந்தார் மாட்டு, இருள் சேர் இரு வினையும் சேரா.
@snapend

@snap[span-100 text-06 text-left]
> இறைவனுடைய உண்மையான புகழை விரும்பி சொல்பவர்கள் அறியாமையை அகற்றி நன்மை தீமைகளை ஒரே அளவில் எதிர் கொள்வார்கள்.
@snapend

---?include=includes/KadavulVazhthu.md

```
பொறிவாயில் ஐந்தவித்தான் பொய்தீர் ஒழுக்க
நெறிநின்றார் நீடுவாழ் வார்.         ( குறள் - 6 )
```

@snap[span-100 text-06 text-left]
> மெய், வாய், கண், மூக்கு, செவி எனும் ஐம்பொறிகளையும் கட்டுப்படுத்திய தூயவனின் உண்மையான ஒழுக்கமுடைய நெறியைப் பின்பற்றி நிற்பவர்களின் புகழ்வாழ்வு நிலையானதாக அமையும்.
@snapend

---?include=includes/KadavulVazhthu.md

```
தனக்குவமை இல்லாதான் தாள்சேர்ந்தார்க் கல்லால்
மனக்கவலை மாற்றல் அரிது.         ( குறள் - 7 )
```

@snap[span-100 text-06 text-left]
> தனக்கு நிகர் இல்லாதவனுடைய திருவடியை பற்றி நடப்பவர்களைத் தவிர , மற்றவர்களின் மனக்கவலை தீர வழியேதுமில்லை.
@snapend

---?include=includes/KadavulVazhthu.md

```
அறவாழி அந்தணன் தாள்சேர்ந்தார்க் கல்லால்
பிறவாழி நீந்தல் அரிது.         ( குறள் - 8 )
```

@snap[span-100 text-06 text-left]
> அந்தணர் என்பதற்குப் பொருள் சான்றோர் என்பதால், அறக்கடலாகவே விளங்கும் அந்த சான்றோரின் அடியொற்றி நடப்பவர்க்கேயன்றி, மற்றவர்களுக்குப் பிறவி துன்பக் கடல்களைக் கடப்பது என்பது எளிதான காரியமல்ல. தர்மத்தை நேசிக்காதவன் பொருளையும் இன்பத்தையும் அடைய முடியாது.
@snapend

---?include=includes/KadavulVazhthu.md

```
கோளில் பொறியின் குணமிலவே எண்குணத்தான்
தாளை வணங்காத் தலை.         ( குறள் - 9 )
```

@snap[span-100 text-gray text-left text-04]
**தொடரமைப்பு:** கோள் இல் பொறியின் குணம் இலவே, எண் குணத்தான் தாளை வணங்காத் தலை.
@snapend

@snap[span-100 text-06 text-left]
> உடல், கண், காது, மூக்கு, வாய் எனும் ஐம்பொறிகள் இருந்தும், அவை இயங்காவிட்டால் அவைகளால் பயன் ஒன்றும் இல்லை. அதேபோல் இறைவனை வணங்காத தலை இருந்தும் பயன் இல்லை.
@snapend

---?include=includes/KadavulVazhthu.md

```
பிறவிப் பெருங்கடல் நீந்துவர் நீந்தார்
இறைவன் அடிசேரா தார்.         ( குறள் - 10 )
```

@snap[span-100 text-gray text-left text-04]
**தொடரமைப்பு:** இறைவன் அடி (சேர்ந்தார்) பிறவி பெரு கடல் நீந்துவர், சேராதார் நீந்தார்.
@snapend

@snap[span-100 text-06 text-left]
> இறைவனுடைய திருவடியை சேர்ந்தவர்களே பிறவி எனும் பெருங்கடலை நீந்துவர். அவ்வாறின்றி மாறி நினைப்பவர்கள் பிறவி பெருங்கடலை நீந்தமுடியாமல் தவிக்க நேரிடும்.
@snapend

---?include=includes/AranValiyuruthal.md

@snap[span-100 text-06 text-left]
``` 
அறம் செய விரும்பு - ( ஔவையார் )
```
@snapend

@snap[span-100 text-06 text-left]
```
அறமிதென்றும் யாம் மறமிதென்றுமே
அறிகிலாத போது - யாம்
அறிகிலாத போது - தமிழ்
இறைவனாரின் திருக்குறளிலே ஒரு சொல்
இயம்பிக் காட்ட மாட்டாயா? - ( பாரதிதாசன் )
```
@snapend

---?include=includes/AranValiyuruthal.md

```
சிறப்பீனும் செல்வமும் ஈனும் அறத்தினூஉங்கு
ஆக்கம் எவனோ உயிர்க்கு.         ( குறள் - 31 )
```

@snap[span-100 text-left text-04]
தொடரமைப்பு: சிறப்பு ஈனும் செல்வமும் ஈனும் உயிர்க்கு அறத்தின் ஊங்கு ஆக்கம் எவன்
@snapend

@snap[span-100 text-06 text-left]
> அறம் சிறப்பையும், செல்வத்தையும் அளிக்கும். ஆகையால் அத்தகைய அறத்தை விட உயிர்க்கு மேன்மையானது வேறு எதுவும் இல்லை.
@snapend

---?include=includes/AranValiyuruthal.md

```
அறத்தினூஉங் காக்கமும் இல்லை அதனை
மறத்தலின் ஊங்கில்லை கேடு.        ( குறள் - 32 )
```

@snap[span-100 text-left text-04]
தொடரமைப்பு: அறத்தின் ஊங்கு ஆக்கமும் இல்லை, அதனை மறத்தலின் ஊங்கு கேடும் இல்லை.
@snapend

@snap[span-100 text-06 text-left]
> அறம் செய்வதை விட நன்மையும் இல்லை. அதைச் செய்ய மறப்பதைவிட கெடுதியும் இல்லை.
@snapend

---?include=includes/AranValiyuruthal.md

```
ஒல்லும் வகையான் அறவினை ஓவாதே
செல்லும்வாய் எல்லாஞ் செயல்.         ( குறள் - 33 )
```

@snap[span-100 text-06 text-left]
> நம்மால் எந்தெந்த வகையில் அறம் செய்ய முடியுமோ, அவ்வகையில் இடைவிடாது நாம் செய்யும் எல்லா செயலையும் அறத்துடன் செய்ய வேண்டும்.
@snapend

---?include=includes/AranValiyuruthal.md

```
மனத்துக்கண் மாசிலன் ஆதல் அனைத்தறன்
ஆகுல நீர பிற		( குறள் - 34)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: மனத்துக்கண் மாசு இலன் ஆதல் அனைத்து அறன், 
பிற ஆகுல நீர.
@snapend

@snap[span-100 text-06 text-left]

> மனத்தூய்மையே அறம். அதை விடுத்து பிறர் அறிய செய்யப்படும் செயல்கள் ஆரவார தன்மை உடையவை.

@snapend

---?include=includes/AranValiyuruthal.md

```
அழுக்கா றவாவெகுளி இன்னாச்சொல் நான்கும்
இழுக்கா இயன்ற தறம்		( குறள் - 35)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: அழுக்காறு அவாவெகுளி இன்னாச்சொல் நான்கும்
இழுக்கா இயன்றது அறம்.
@snapend

@snap[span-100 text-06 text-left]

> பொறாமை , பேராசை, கோபம் , இன்னாச்சொல் இந்த நான்கு குணங்களையும் நீக்கிவிட்டு வருவது அறம்.

@snapend

---?include=includes/AranValiyuruthal.md

```
அன்றறிவாம் என்னா தறஞ்செய்க மற்றது
பொன்றுங்கால் பொன்றாத் துணை		( குறள் - 36)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: அன்று அறிவாம் என்னாது அறம் செய்க மற்று அது
பொன்றும் கால் பொன்றாத் துணை.
@snapend

@snap[span-100 text-06 text-left]

> பிறகு செய்யலாம் என்று இல்லாமல் அறத்தை நாள்தோறும் செய்க. அவ்வாறு செய்த அறம் , உயிர் இந்த உடலை விட்டு பிரிந்து செல்லும் காலத்தில் உயிருக்கு அழிவில்லாத துணையாக வரும்.

@snapend

```
அறத்தா றிதுவென வேண்டா சிவிகை
பொறுத்தானோ டூர்ந்தான் இடை		( குறள் - 37)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: அறத்து ஆறு இது என வேண்டா சிவிகை பொறுத்தானோடு ஊர்ந்தான் இடை.
@snapend

@snap[span-100 text-06 text-left]

> அறவழியில் நடப்பவர்கள் பல்லக்கில் உட்கார்ந்து செல்பவர்களைப் போல வாழ்க்கையில் வரும் இன்ப துன்பங்கள் இரண்டையும் ஒரே அளவில் கருதி மகிழ்வுடன் வாழ்வார்கள். தீய வழியில் நடப்பவர்கள் பல்லக்கை தூக்கி சுமப்பவர்களை போல இன்பத்திலும் அமைதி கொள்ளாமல் துன்பத்தையும் தாங்கி கொள்ள முடியாமல் வாழ்வையே சுமையாக கருதுவார்கள்.

@snapend

```
வீழ்நாள் படாஅமை நன்றாற்றின் அஃதொருவன்
வாழ்நாள் வழியடைக்கும் கல்		( குறள் - 38)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: வீழ்நாள் படாமை நன்று ஆற்றின் அஃது ஒருவன்
வாழ்நாள் வழி அடைக்கும் கல்.
@snapend

@snap[span-100 text-06 text-left]

> ஒரு நாள் கூட வீண் அடிக்காமல் அறம் செய்பவர்களுக்கு அந்த அறம், அவர்களுக்கு வரும் மறுபிறப்பை அடைக்கும் கல்லாக ஆகும்.

@snapend

```
அறத்தான் வருவதே இன்பமற் றெல்லாம்
புறத்த புகழும் இல		( குறள் - 39)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: அறத்தான் வருவதே இன்பம் மற்று எல்லாம்
புறத்த புகழும் இல.
@snapend

@snap[span-100 text-06 text-left]

> அறவழியில் வருவதே இன்பம். தீயவழியில் வருவது துன்பமே, அது புகழும் இல்லை.

@snapend

செயற்பால தோரும் அறனே ஒருவற்கு 		
உயற்பால தோரும் பழி	( குறள் - 40)

தொடரமைப்பு: ஒருவற்குச் செயற்பால தோரும் அறனே, உயற்பால தோரும் பழி.

@snap[span-100 text-06 text-left]

<div class="meaning" >
இல்வாழ்க்கையை நல்ல முறையில் நடத்துபவன் தனது பெற்றோர், 
படிக்கும் பருவத்தில் உள்ளவர்கள்,  மற்றும் துறவி 
ஆகிய மூவருக்கும் நல்ல துணையாக இருப்பான்.
</div>

@snapend

---?include=includes/Ilvaazhkkai.md

```
துறந்தார்க்கும் துவ்வா தவர்க்கும் இறந்தார்க்கும்
இல்வாழ்வான் என்பான் துணை		( குறள் - 42)

```

@snap[span-100 text-06 text-left]

<div class="meaning" >

தன்னுடைய கடமையிலிருந்து தவறிய குடும்பத்தினரால் கைவிடப்பட்டவர்களுக்கும், 
வறுமையில் இருப்பவர்களுக்கும், தன்னிடம் வந்து இறந்தவருக்கும் 
இல்லறத்தான் துணையாக இருப்பான்.

</div>

@snapend

---?include=includes/Ilvaazhkkai.md

```
தென்புலத்தார் தெய்வம் விருந்தொக்கல் தானென்றாங்கு
ஐம்புலத்தா றோம்பல் தலை		( குறள் - 43)

```

@snap[span-100 text-left text-04]
**தொடரமைப்பு:** தென்புலத்தார் தெய்வம் விருந்து ஒக்கல் தான் என்றாங்கு 
ஐம்புலத்தாறு ஓம்பல் தலை.

@snapend

@snap[span-100 text-06 text-left]

<div class="meaning" >
தனது மூதாதைகள், குலதெய்வம், விருந்தினர், உறவினர்கள் 
ஆகியோரை முறையாகக் கவனித்துவிட்டு, 
தன்னையும் கவனிப்பதே இல்லறத்தான் சிறப்பு.
</div>

@snapend

```
பழியஞ்சிப் பாத்தூண் உடைத்தாயின் வாழ்க்கை
வழியெஞ்சல் எஞ்ஞான்றும் இல்		( குறள் - 44)

```

@snap[span-100 text-left text-04]
**தொடரமைப்பு:**  பழி அஞ்சிப் பாத்தூண் வாழ்க்கை உடைத்தாயின்
வழி எஞ்ஞான்றும் எஞ்சல் இல்.
@snapend

@snap[span-100 text-06 text-left]

> பொருள் சம்பாதிக்கும்பொழுது பழிபாவங்களுக்குப் பயந்து சம்பாதித்து, அந்த பொருளை மற்றவருக்கு பகிர்ந்து உண்பவனது சந்ததி **புகழுடன்** வாழும்.

@snapend

---?include=includes/Ilvaazhkkai.md

```
அன்பும் அறனும் உடைத்தாயின் இல்வாழ்க்கை
பண்பும் பயனும் அது		( குறள் - 45)

```

@snap[span-100 text-left text-04]
**தொடரமைப்பு:**  இல்வாழ்க்கை அன்பும் அறனும் உடைத்தாயின் 
அது பண்பும் பயனும்.
@snapend

@snap[span-100 text-06 text-left]

> அன்பு பண்பாக உள்ள இல்லறத்தில் அறம் அதன் பயனாக வரும்.

@snapend

---?include=includes/Ilvaazhkkai.md

```
அறத்தாற்றின் இல்வாழ்க்கை ஆற்றின் புறத்தாற்றில்
போஒய்ப் பெறுவ தெவன்		( குறள் - 46)

```

@snap[span-100 text-left text-04]
**தொடரமைப்பு:**  இல்வாழ்க்கை அறத்தாற்றின் ஆற்றின், புறத்தாற்றின் 
போஓய்ப் பெறுவது எவன்.
@snapend

@snap[span-100 text-06 text-left]

> அறவழியில் மனைவி, பிள்ளைகளுடன் இல்வாழ்க்கை நடத்தினால் அதுவே பெரும் பேறு. துறவு வாழ்க்கையை விட இல்லறமே மேலானது.

@snapend

---?include=includes/Ilvaazhkkai.md

```
இயல்பினான் இல்வாழ்க்கை வாழ்பவன் என்பான்
முயல்வாருள் எல்லாம் தலை		( குறள் - 47)

```

@snap[span-100 text-left text-04]
**தொடரமைப்பு:**  இல்வாழ்க்கை இயல்பினான் வாழ்பவன் என்பான்
முயல்வாருள் எல்லாம் தலை.
@snapend

@snap[span-100 text-06 text-left]

> இல்வாழ்க்கையை அதன் இயல்புடன் வாழ்பவன் துறவுக்கு முயல்பவர்களை விட மேலானவன்.

@snapend

---?include=includes/Ilvaazhkkai.md

```
ஆற்றின் ஒழுக்கி அறனிழுக்கா இல்வாழ்க்கை
நோற்பாரின் நோன்மை உடைத்து		( குறள் - 48)

```

@snap[span-100 text-left text-04]
**தொடரமைப்பு:**  ஆற்றின் ஒழுக்கி அறன் இழுக்கா இல்வாழ்க்கை
நோற்பாரின் நோன்மை உடைத்து.
@snapend

@snap[span-100 text-06 text-left]

> இல்வாழ்க்கையில் நெறிதவறாமல் நடந்து, மற்றவர்களும் அறநெறியில் நடக்க உதவுபவன் தவம் செய்பவர்களை விட கடமையும் சகிப்புத்தன்மையும் உடையவனாகிறான்.

@snapend

---?include=includes/Ilvaazhkkai.md

```
அறன்எனப் பட்டதே இல்வாழ்க்கை அஃதும்
பிறன்பழிப்ப தில்லாயின் நன்று		( குறள் - 49)

```

@snap[span-100 text-left text-04]
**தொடரமைப்பு:**  அறன் எனப்பட்டது இல்வாழ்க்கையே 
அஃதும் பிறன்பழிப்பது இல்லாயின் நன்று.
@snapend

@snap[span-100 text-06 text-left]

> நூல்களால் அறம் என்று சொல்லப்படுகின்ற குடும்ப வாழ்க்கையே துறவை விட மேலானது. துறவறமும் மற்றவர்களால் பழிக்க படாமல் இருந்தால் நல்லது.

@snapend

---?include=includes/Ilvaazhkkai.md

```
வையத்துள் வாழ்வாங்கு வாழ்பவன் வான்உறையும்
தெய்வத்துள் வைக்கப் படும்		( குறள் - 50)

```

@snap[span-100 text-left text-04]
**தொடரமைப்பு:**  வாழ்வாங்கு வையத்துள் வாழ்பவன் வான் உறையும் 
தெய்வத்துள் வைக்கப் படும்.

@snapend

@snap[span-100 text-06 text-left]

> அறத்துடன் கூடிய இல்லறம் நடத்தி இவ்வுலகில் வாழ்பவன் வானத்துத் தெய்வங்களுக்கு நிகரானவன்.

@snapend

@snap[span-100 text-06 text-left]


<div class="conclusion" >

இல்லறத்தில் வாழ்பவன்:

* தன் மனைவி, பிள்ளைகளிடத்து அன்புடனும் ஒற்றுமையுடனும் இருக்க வேண்டும். 
* பெற்றோருக்கு துணையாக இருக்க வேண்டும். 
* மாணவ பருவத்தில் உள்ளவருக்கு  உதவியாக இருக்க வேண்டும். 
* துறவிகள் அவர்கள் நெறியில் நடக்க உதவ வேண்டும். 
* குடும்பத்தினரால் கைவிடபட்டவருக்கும், வறுமையில் இருப்பவர்களுக்கும் உதவ வேண்டும். 
* யாருமில்லாமல் தன்னிடம் வந்து இறந்தவரின் இறுதிக் கடனை செய்ய வேண்டும். 
* மூதாதைகளையும், குலதெய்வத்தையும் வணங்க வேண்டும்.
* விருந்தினரை உபசரிக்க வேண்டும். உறவினரிடம் ஒற்றுமையாக இருக்க வேண்டும். 
* இத்துடன் தன்னையும் காத்து கொள்ள வேண்டும்.  
* இதற்கான பொருளை பழி பாவங்களுக்கு பயந்து சேர்க்க வேண்டும். 
* இவ்வாறு வாழ்பவன் முனிவர்களை விட மேலானவன். 
* வானில் வாழும் தெய்வங்களுக்கு நிகரானவன்.

</div>

> இல்லறத்திற்கே உரிய நல்ல பண்புகளைப் பெற்றவளாய் இருப்பதுடன் கணவனுடைய வருவாய்க்கு ஏற்றபடி செலவு செய்து வாழ்பவளே நல்ல வாழ்க்கைத்துணை.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
மனைமாட்சி இல்லாள்கண் இல்லாயின் வாழ்க்கை
எனைமாட்சித் தாயினும் இல்		( குறள் - 52)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: மனை மாட்சி இல்லாள் கண் இல் ஆயின் வாழ்க்கை 

எனை மாட்சித்து ஆயினும் இல்.
@snapend

@snap[span-100 text-06 text-left]

> குடும்ப வாழ்க்கைக்கு ஏற்ற நல்ல குணம் ,நல்ல செய்கை மனைவியிடம் இல்லையெனில் வேறு சிறப்புகள் எத்தனை இருந்தாலும் அதனால் பயனில்லை.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
இல்லதென் இல்லவள் மாண்பானால் உள்ளதென்
இல்லவள் மாணாக் கடை		( குறள் - 53)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: இல்லவள் மாண்பானால் இல்லதென் 

இல்லவள் மாணாக்கடை உள்ளதென்.
@snapend

@snap[span-100 text-06 text-left]

> இல்வாழ்க்கைக்கு உரிய நற்குணங்கள் மனைவியிடத்தில் இருந்தால் அந்த குடும்பத்தில் இல்லாதது எதுவும் இல்லை. அக்குணங்கள் இல்லாமல் போனால் அங்கே எல்லாம் இருந்தாலும் எதுவுமே இல்லாதது போலாம்.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
பெண்ணின் பெருந்தக்க யாவுள கற்பென்னும்
திண்மைஉண் டாகப் பெறின்		( குறள் - 54)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: கற்பு என்னும் திண்மை உண்டாகப் பெறின் 

பெண்ணின் பெருந்தக்க யாவுள.
@snapend

@snap[span-100 text-06 text-left]

> கற்பு என்னும் மனஉறுதி மட்டும் இருந்தால் அதைவிட பெருமையானது உலகில் வேறு எதுவுமில்லை.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
தெய்வம் தொழாஅள் கொழுநற் றொழுதெழுவாள்
பெய்யெனப் பெய்யும் மழை		( குறள் - 55)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: தெய்வம் தொழாள் கொழுநன் தொழுது எழுவாள் 

பெய் என பெய்யும் மழை.
@snapend

@snap[span-100 text-06 text-left]

> தன் கணவனையே எல்லாமுமாக எண்ணி கற்புடன் வாழும் பெண்ணுக்கு இந்த இயற்கையே கட்டுப்படும்.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
தற்காத்துத் தற்கொண்டாற் பேணித் தகைசான்ற
சொற்காத்துச் சோர்விலாள் பெண்		( குறள் - 56)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: தன் காத்து தன் கொண்டான் பேணித் தகை சான்ற 

சொல் காத்துச் சோர்வு இலாள் பெண்.
@snapend

@snap[span-100 text-06 text-left]

தனது பெருமைமிக்க கற்பினைக் காத்து கணவனின் நலன்களைப் பேணிக் காத்து, தன் குடும்பத்தின் புகழையும் ஒரு பெண் கடைசி வரை காக்க வேண்டும்.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
சிறைகாக்கும் காப்பெவன் செய்யும் மகளிர்
நிறைகாக்கும் காப்பே தலை		( குறள் - 57)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: மகளிர் சிறை காக்கும் காப்பு எவன் செய்யும் 

நிறை காக்கும் காப்பே தலை. 
@snapend

@snap[span-100 text-06 text-left]

> ஒரு பெண்ணைக் கணவனோ, பெற்றோர்களோ வீட்டிலேயே அடைத்து வைத்தால் மட்டுமே கற்பைக் காப்பாற்றி வீட்டா முடியாது. அவளது கற்புணர்ச்சியே அவளுக்கு சிறந்த காவலாக இருக்கமுடியும்.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
பெற்றாற் பெறின்பெறுவர் பெண்டிர் பெருஞ்சிறப்புப்
புத்தேளிர் வாழும் உலகு		( குறள் - 58)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: பெண்டிர் பெற்றான் பேறின் 

புத்தேளிர் வாழும் உலகு பெரும் சிறப்புப் பெறுவர்.
@snapend

@snap[span-100 text-06 text-left]

> கணவனைப் போற்றி கடமையைச் செய்யப் பெற்றால் , அந்த பெண் தேவர்கள் வாழும் பெரிய சிறப்பினையுடைய உலகத்தைப் பெறுவர்.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
புகழ்புரிந் தில்லிலோர்க் கில்லை இகழ்வார்முன்
ஏறுபோல் பீடு நடை		( குறள் - 59)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: புகழ் புரிந்த இல் இலோர்க்கு இல்லை இகழ்வார்முன் 

ஏறு போல் பீடு நடை
@snapend

@snap[span-100 text-06 text-left]

> குடும்ப புகழை காக்க விரும்பும் மனைவி இல்லாதவருக்கு மக்கள் கூடிய சாலையில் தலைநிமிர்ந்து நடக்க முடியாது.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
மங்கலம் என்ப மனைமாட்சி மற்றதன்
நன்கலம் நன்மக்கட் பேறு		( குறள் - 60)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: மங்கலம் என்ப மனை மாட்சி 

அதன் நன் கலம் நன் மக்கள் பேறு.
@snapend

@snap[span-100 text-06 text-left]

> ஒருவனுக்கு நல்ல பண்புகள் உள்ள மனைவியே அழகு. அந்த அழகிற்கு ஏற்ற அணிகலன்கள் நல்லப் பிள்ளைகளை பெறுவதே.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

@snap[span-100 text-06 text-left]

> இல்லறத்திற்கு உரிய பண்புகளாவன:

      விருந்தினரை உபசரித்தல்.

       உறவினருடன் ஒற்றுமையாக இருத்தல்.

       வறுமையில் இருப்பவருக்கு உதவுதல்

       நல்ல சமையல் திறன் 

       கணவன் செய்யும் அறச்செயல்களுக்கு துணையாய் இருத்தல் முதலியன 

இதனை தன்னிடம் இருப்பதை கொண்டு வருமானத்துக்குள் செய்யவேண்டும். 

அந்த பெண் மனவுறுதியோடு தன்னை தானே கற்பு நெறியில் காத்துக்கொண்டு தன்  கணவனையும்,தன குடும்பத்து புகழையும் காக்கவேண்டும். கணவனுக்கு உண்மையானவளாக கற்புநெறியில் வாழும் பெண்ணிற்கு இந்த இயற்கையே கட்டுப்படும். இல்லறத்திற்கு அழகு வாழ்க்கைத்துணையின்  பண்புதான்.

அந்த அழகிற்கு அணிகலன் நல்ல பிள்ளைகள். அப்படிப்பட்ட பண்புள்ள பெண்ணை துணையாக கொண்டவர் ஊரார் முன் தலைநிமிர்ந்து வாழலாம்.         

@snapend
