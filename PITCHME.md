## திருக்குறள் அறிமுகம்

@img[width=300, height=350 , shadow](assets/img/valluvar.png)

2000 ஆண்டுகளுக்கு மேல் பழமையான இந்த நூலை இயற்றியவர் **திருவள்ளுவர்.**

---

### திருக்குறளின் உண்மையான பெயர் முப்பால். 
 
அறத்தை வலியுறுத்துவதால், இந்நூல் **அற நூல்** என்று சொல்லப் படுகிறது.

---
### இதன் வேறு பெயர்கள்

* உலகப்  பொது மறை
* பொய்யா மொழி
* உத்திர வேதம்
* தெய்வ  நூல் 
* வான் மறை 
* ஈரடி நூல் 
* தமிழ் மறை 

---

### திருக்குறளின் பயன் என்ன?

அறத்தை தெரிந்து கொண்டு அதன் படி வாழ்வது.

---

### அறம் என்றால் என்ன?

```
மனத்துக்கண் மாசிலன் ஆதல் அனைத்தறன்
ஆகுல நீர பிற. 
```

> மனத்தூய்மையே **அறம்**.

---

### மனத்தூய்மை என்றால் என்ன?

```
அழுக்காறு அவாவெகுளி இன்னாச்சொல் நான்கும்
இழுக்கா இயன்றது அறம்.
```

> பொறாமை, பேராசை, கோபம், புண்படுத்தும் சொல் ஆகியவை
> இல்லாமல் வாழ்வதே **அறம்** .

---

### ஏன் அற வழியில் வாழ வேண்டும் ?

```
அறத்தான் வருவதே இன்பம் மற்றெல்லாம்
புறத்த புகழும் இல.
```

> நாம் அற வழியில்  ஈட்டுகின்ற பொருளும் புகழுமே  நமக்கு  இன்பம்  தரும்.
---

### திருக்குறள் வரிசை படுத்தப்பட்டுள்ள முறை 

@img[width=450, height=450 , shadow](assets/img/thirukural-organization1.png)

---?include=includes/KadavulVazhthu.md

```
அகர முதல எழுத்தெல்லாம் ஆதி
பகவன் முதற்றே உலகு.            ( குறள் - 1 )
```

@snap[span-100 text-gray text-left text-04]
**தொடரமைப்பு:** எழுத்தெல்லாம் அகர முதல, உலகம் ஆதி பகவன் முதற்றே
@snapend

@snap[span-100 text-06 text-left]
> அகரம் எழுத்துக்களுக்கு முதன்மை; ஆதிபகவன், உலகில் வாழும் உயிர்களுக்கு முதன்மை.
@snapend

---?include=includes/KadavulVazhthu.md

```
கற்றதனால் ஆய பயனென்கொல் வாலறிவன்
நற்றாள் தொழாஅர் எனின்.        ( குறள் - 2 )
```

@snap[span-100 text-gray text-left text-04]
**தொடரமைப்பு:** வாலறிவன் நற்றாள் தொழாஅர் எனின், கற்றதனால் ஆய பயன் என்கொல்?
@snapend

@snap[span-100 text-06 text-left]
> இறைவனுடைய திருவடியை வணங்கும் பண்பு இல்லாவிடில் என்னதான் ஒருவர் கற்றிருந்தாலும் அதனால் என்ன பயன்? ஒன்றுமில்லை.
@snapend

---?include=includes/KadavulVazhthu.md

```
மலர்மிசை ஏகினான் மாணடி சேர்ந்தார்
நிலமிசை நீடுவாழ் வார்.         ( குறள் - 3 )
```

@snap[span-100 text-06 text-left]
> அன்பர்களின் உள்ளமாகிய மலரில் வாழ்பவனாகிய இறைவனின் திருவடியை மனதிலே இடைவிடாது நினைத்து வாழ்பவரின் புகழ் வாழ்வு , உலகில் நெடுங்காலம் நிலைத்து நிற்கும்.
@snapend

---?include=includes/KadavulVazhthu.md

```
வேண்டுதல் வேண்டாமை இலானடி சேர்ந்தார்க்கு
யாண்டும் இடும்பை இல.         ( குறள் - 4 )
```

@snap[span-100 text-06 text-left]
> விருப்பு வெறுப்பற்ற இறைவனையே என்றும் மனதில் நினைப்பவர்களுக்கு அத்தன்மையே ஏற்படுவதால்  அவர்களுக்கு எப்போதும் துன்பம் இல்லை.
@snapend

---?include=includes/KadavulVazhthu.md

```
இருள்சேர் இருவினையும் சேரா இறைவன்
பொருள்சேர் புகழ்புரிந்தார் மாட்டு.        ( குறள் - 5 )
```

@snap[span-100 text-gray text-left text-04]
**தொடரமைப்பு:** இறைவன் பொருள் சேர் புகழ் புரிந்தார் மாட்டு, இருள் சேர் இரு வினையும் சேரா.
@snapend

@snap[span-100 text-06 text-left]
> இறைவனுடைய உண்மையான புகழை விரும்பி சொல்பவர்கள் அறியாமையை அகற்றி நன்மை தீமைகளை ஒரே அளவில் எதிர் கொள்வார்கள்.
@snapend

---?include=includes/KadavulVazhthu.md

```
பொறிவாயில் ஐந்தவித்தான் பொய்தீர் ஒழுக்க
நெறிநின்றார் நீடுவாழ் வார்.         ( குறள் - 6 )
```

@snap[span-100 text-06 text-left]
> மெய், வாய், கண், மூக்கு, செவி எனும் ஐம்பொறிகளையும் கட்டுப்படுத்திய தூயவனின் உண்மையான ஒழுக்கமுடைய நெறியைப் பின்பற்றி நிற்பவர்களின் புகழ்வாழ்வு நிலையானதாக அமையும்.
@snapend

---?include=includes/KadavulVazhthu.md

```
தனக்குவமை இல்லாதான் தாள்சேர்ந்தார்க் கல்லால்
மனக்கவலை மாற்றல் அரிது.         ( குறள் - 7 )
```

@snap[span-100 text-06 text-left]
> தனக்கு நிகர் இல்லாதவனுடைய திருவடியை பற்றி நடப்பவர்களைத் தவிர , மற்றவர்களின் மனக்கவலை தீர வழியேதுமில்லை.
@snapend

---?include=includes/KadavulVazhthu.md

```
அறவாழி அந்தணன் தாள்சேர்ந்தார்க் கல்லால்
பிறவாழி நீந்தல் அரிது.         ( குறள் - 8 )
```

@snap[span-100 text-06 text-left]
> அந்தணர் என்பதற்குப் பொருள் சான்றோர் என்பதால், அறக்கடலாகவே விளங்கும் அந்த சான்றோரின் அடியொற்றி நடப்பவர்க்கேயன்றி, மற்றவர்களுக்குப் பிறவி துன்பக் கடல்களைக் கடப்பது என்பது எளிதான காரியமல்ல. தர்மத்தை நேசிக்காதவன் பொருளையும் இன்பத்தையும் அடைய முடியாது.
@snapend

---?include=includes/KadavulVazhthu.md

```
கோளில் பொறியின் குணமிலவே எண்குணத்தான்
தாளை வணங்காத் தலை.         ( குறள் - 9 )
```

@snap[span-100 text-gray text-left text-04]
**தொடரமைப்பு:** கோள் இல் பொறியின் குணம் இலவே, எண் குணத்தான் தாளை வணங்காத் தலை.
@snapend

@snap[span-100 text-06 text-left]
> உடல், கண், காது, மூக்கு, வாய் எனும் ஐம்பொறிகள் இருந்தும், அவை இயங்காவிட்டால் அவைகளால் பயன் ஒன்றும் இல்லை. அதேபோல் இறைவனை வணங்காத தலை இருந்தும் பயன் இல்லை.
@snapend

---?include=includes/KadavulVazhthu.md

```
பிறவிப் பெருங்கடல் நீந்துவர் நீந்தார்
இறைவன் அடிசேரா தார்.         ( குறள் - 10 )
```

@snap[span-100 text-gray text-left text-04]
**தொடரமைப்பு:** இறைவன் அடி (சேர்ந்தார்) பிறவி பெரு கடல் நீந்துவர், சேராதார் நீந்தார்.
@snapend

@snap[span-100 text-06 text-left]
> இறைவனுடைய திருவடியை சேர்ந்தவர்களே பிறவி எனும் பெருங்கடலை நீந்துவர். அவ்வாறின்றி மாறி நினைப்பவர்கள் பிறவி பெருங்கடலை நீந்தமுடியாமல் தவிக்க நேரிடும்.
@snapend

---?include=includes/AranValiyuruthal.md

@snap[span-100 text-06 text-left]
``` 
அறம் செய விரும்பு - ( ஔவையார் )
```
@snapend

@snap[span-100 text-06 text-left]
```
அறமிதென்றும் யாம் மறமிதென்றுமே
அறிகிலாத போது - யாம்
அறிகிலாத போது - தமிழ்
இறைவனாரின் திருக்குறளிலே ஒரு சொல்
இயம்பிக் காட்ட மாட்டாயா? - ( பாரதிதாசன் )
```
@snapend

---?include=includes/AranValiyuruthal.md

```
சிறப்பீனும் செல்வமும் ஈனும் அறத்தினூஉங்கு
ஆக்கம் எவனோ உயிர்க்கு.         ( குறள் - 31 )
```

@snap[span-100 text-left text-04]
தொடரமைப்பு: சிறப்பு ஈனும் செல்வமும் ஈனும் உயிர்க்கு அறத்தின் ஊங்கு ஆக்கம் எவன்
@snapend

@snap[span-100 text-06 text-left]
> அறம் சிறப்பையும், செல்வத்தையும் அளிக்கும். ஆகையால் அத்தகைய அறத்தை விட உயிர்க்கு மேன்மையானது வேறு எதுவும் இல்லை.
@snapend

---?include=includes/AranValiyuruthal.md

```
அறத்தினூஉங் காக்கமும் இல்லை அதனை
மறத்தலின் ஊங்கில்லை கேடு.        ( குறள் - 32 )
```

@snap[span-100 text-left text-04]
தொடரமைப்பு: அறத்தின் ஊங்கு ஆக்கமும் இல்லை, அதனை மறத்தலின் ஊங்கு கேடும் இல்லை.
@snapend

@snap[span-100 text-06 text-left]
> அறம் செய்வதை விட நன்மையும் இல்லை. அதைச் செய்ய மறப்பதைவிட கெடுதியும் இல்லை.
@snapend

---?include=includes/AranValiyuruthal.md

```
ஒல்லும் வகையான் அறவினை ஓவாதே
செல்லும்வாய் எல்லாஞ் செயல்.         ( குறள் - 33 )
```

@snap[span-100 text-06 text-left]
> நம்மால் எந்தெந்த வகையில் அறம் செய்ய முடியுமோ, அவ்வகையில் இடைவிடாது நாம் செய்யும் எல்லா செயலையும் அறத்துடன் செய்ய வேண்டும்.
@snapend

---?include=includes/AranValiyuruthal.md

```
மனத்துக்கண் மாசிலன் ஆதல் அனைத்தறன்
ஆகுல நீர பிற		( குறள் - 34)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: மனத்துக்கண் மாசு இலன் ஆதல் அனைத்து அறன், 
பிற ஆகுல நீர.
@snapend

@snap[span-100 text-06 text-left]

> மனத்தூய்மையே அறம். அதை விடுத்து பிறர் அறிய செய்யப்படும் செயல்கள் ஆரவார தன்மை உடையவை.

@snapend

---?include=includes/AranValiyuruthal.md

```
அழுக்கா றவாவெகுளி இன்னாச்சொல் நான்கும்
இழுக்கா இயன்ற தறம்		( குறள் - 35)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: அழுக்காறு அவாவெகுளி இன்னாச்சொல் நான்கும்
இழுக்கா இயன்றது அறம்.
@snapend

@snap[span-100 text-06 text-left]

> பொறாமை , பேராசை, கோபம் , இன்னாச்சொல் இந்த நான்கு குணங்களையும் நீக்கிவிட்டு வருவது அறம்.

@snapend

---?include=includes/AranValiyuruthal.md

```
அன்றறிவாம் என்னா தறஞ்செய்க மற்றது
பொன்றுங்கால் பொன்றாத் துணை		( குறள் - 36)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: அன்று அறிவாம் என்னாது அறம் செய்க மற்று அது
பொன்றும் கால் பொன்றாத் துணை.
@snapend

@snap[span-100 text-06 text-left]

> பிறகு செய்யலாம் என்று இல்லாமல் அறத்தை நாள்தோறும் செய்க. அவ்வாறு செய்த அறம் , உயிர் இந்த உடலை விட்டு பிரிந்து செல்லும் காலத்தில் உயிருக்கு அழிவில்லாத துணையாக வரும்.

@snapend

```
துறந்தார்க்கும் துவ்வா தவர்க்கும் இறந்தார்க்கும்
```

<div class="meaning" >

தன்னுடைய கடமையிலிருந்து தவறிய குடும்பத்தினரால் கைவிடப்பட்டவர்களுக்கும், 
வறுமையில் இருப்பவர்களுக்கும், தன்னிடம் வந்து இறந்தவருக்கும் 
இல்லறத்தான் துணையாக இருப்பான்.

```
தென்புலத்தார் தெய்வம் விருந்தொக்கல் தானென்றாங்கு
```

ஐம்புலத்தாறு ஓம்பல் தலை.

தனது மூதாதைகள், குலதெய்வம், விருந்தினர், உறவினர்கள் 
ஆகியோரை முறையாகக் கவனித்துவிட்டு, 
தன்னையும் கவனிப்பதே இல்லறத்தான் சிறப்பு.

@snap[span-100 text-left text-04]
**தொடரமைப்பு:**  இல்வாழ்க்கை அன்பும் அறனும் உடைத்தாயின் 
அது பண்பும் பயனும்.
@snapend

@snapend

```
அறத்தாற்றின் இல்வாழ்க்கை ஆற்றின் புறத்தாற்றில்
போஒய்ப் பெறுவ தெவன்		( குறள் - 46)
```

@snap[span-100 text-left text-04]

போஓய்ப் பெறுவது எவன்.

முயல்வாருள் எல்லாம் தலை.

@snap[span-100 text-06 text-left]


<div class="conclusion" >

இல்லறத்தில் வாழ்பவன்:

* தன் மனைவி, பிள்ளைகளிடத்து அன்புடனும் ஒற்றுமையுடனும் இருக்க வேண்டும். 
* பெற்றோருக்கு துணையாக இருக்க வேண்டும். 
* மாணவ பருவத்தில் உள்ளவருக்கு  உதவியாக இருக்க வேண்டும். 
* துறவிகள் அவர்கள் நெறியில் நடக்க உதவ வேண்டும். 
* குடும்பத்தினரால் கைவிடபட்டவருக்கும், வறுமையில் இருப்பவர்களுக்கும் உதவ வேண்டும். 
* யாருமில்லாமல் தன்னிடம் வந்து இறந்தவரின் இறுதிக் கடனை செய்ய வேண்டும். 
* மூதாதைகளையும், குலதெய்வத்தையும் வணங்க வேண்டும்.
* விருந்தினரை உபசரிக்க வேண்டும். உறவினரிடம் ஒற்றுமையாக இருக்க வேண்டும். 
* இத்துடன் தன்னையும் காத்து கொள்ள வேண்டும்.  
* இதற்கான பொருளை பழி பாவங்களுக்கு பயந்து சேர்க்க வேண்டும். 
* இவ்வாறு வாழ்பவன் முனிவர்களை விட மேலானவன். 
* வானில் வாழும் தெய்வங்களுக்கு நிகரானவன்.

</div>

> இல்லறத்திற்கே உரிய நல்ல பண்புகளைப் பெற்றவளாய் இருப்பதுடன் கணவனுடைய வருவாய்க்கு ஏற்றபடி செலவு செய்து வாழ்பவளே நல்ல வாழ்க்கைத்துணை.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
மனைமாட்சி இல்லாள்கண் இல்லாயின் வாழ்க்கை
எனைமாட்சித் தாயினும் இல்		( குறள் - 52)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: மனை மாட்சி இல்லாள் கண் இல் ஆயின் வாழ்க்கை 
எனை மாட்சித்து ஆயினும் இல்.
@snapend

@snap[span-100 text-08 text-left]

> குடும்ப வாழ்க்கைக்கு ஏற்ற நல்ல குணம், நல்ல செய்கை மனைவியிடம் இல்லையெனில் வேறு சிறப்புகள் எத்தனை இருந்தாலும் அதனால் பயனில்லை.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
இல்லதென் இல்லவள் மாண்பானால் உள்ளதென்
இல்லவள் மாணாக் கடை		( குறள் - 53)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: இல்லவள் மாண்பானால் இல்லதென் 
இல்லவள் மாணாக்கடை உள்ளதென்.
@snapend

@snap[span-100 text-08 text-left]

> இல்வாழ்க்கைக்கு உரிய நற்குணங்கள் மனைவியிடத்தில் இருந்தால் அந்த குடும்பத்தில் இல்லாதது எதுவும் இல்லை. அக்குணங்கள் இல்லாமல் போனால் அங்கே எல்லாம் இருந்தாலும் எதுவுமே இல்லாதது போலாம்.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
பெண்ணின் பெருந்தக்க யாவுள கற்பென்னும்
திண்மைஉண் டாகப் பெறின்		( குறள் - 54)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: கற்பு என்னும் திண்மை உண்டாகப் பெறின் 
பெண்ணின் பெருந்தக்க யாவுள.
@snapend

@snap[span-100 text-08 text-left]

> கற்பு என்னும் மனஉறுதி ஒரு பெண்ணிடம் இருந்தால் அதைவிட பெருமையானது  வேறு எதுவுமில்லை.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
தெய்வம் தொழாஅள் கொழுநற் றொழுதெழுவாள்
பெய்யெனப் பெய்யும் மழை		( குறள் - 55)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: தெய்வம் தொழாள் கொழுநன் தொழுது எழுவாள் 
பெய் என பெய்யும் மழை.
@snapend

@snap[span-100 text-08 text-left]

> தன் கணவனையே எல்லாமுமாக எண்ணி கற்புடன் வாழும் பெண்ணுக்கு இந்த இயற்கையே கட்டுப்படும்.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
தற்காத்துத் தற்கொண்டாற் பேணித் தகைசான்ற
சொற்காத்துச் சோர்விலாள் பெண்		( குறள் - 56)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: தன் காத்து தன் கொண்டான் பேணித் தகை சான்ற 
சொல் காத்துச் சோர்வு இலாள் பெண்.
@snapend

@snap[span-100 text-08 text-left]

> தனது பெருமைமிக்க கற்பினைக் காத்து கணவனின் நலன்களைப் பேணிக் காத்து, தன் குடும்பத்தின் புகழையும் ஒரு பெண் கடைசி வரை காக்க வேண்டும்.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
சிறைகாக்கும் காப்பெவன் செய்யும் மகளிர்
நிறைகாக்கும் காப்பே தலை		( குறள் - 57)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: மகளிர் சிறை காக்கும் காப்பு எவன் செய்யும் 
நிறை காக்கும் காப்பே தலை. 
@snapend

@snap[span-100 text-08 text-left]

> ஒரு பெண்ணைக் கணவனோ, பெற்றோர்களோ வீட்டிலேயே அடைத்து வைத்தால் மட்டுமே கற்பைக் காப்பாற்றி விட முடியாது. அவளது கற்புணர்ச்சியே அவளுக்கு சிறந்த காவலாக இருக்கமுடியும்.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
பெற்றாற் பெறின்பெறுவர் பெண்டிர் பெருஞ்சிறப்புப்
புத்தேளிர் வாழும் உலகு		( குறள் - 58)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: பெண்டிர் பெற்றான் பெறின் 
புத்தேளிர் வாழும் உலகு பெரும் சிறப்புப் பெறுவர்.
@snapend

@snap[span-100 text-08 text-left]

> கணவனைப் போற்றி கடமையைச் செய்யப் பெற்றால், அந்த பெண் தேவர்கள் வாழும் பெரிய சிறப்பினையுடைய உலகத்தைப் பெறுவர்.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
புகழ்புரிந் தில்லிலோர்க் கில்லை இகழ்வார்முன்
ஏறுபோல் பீடு நடை		( குறள் - 59)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: புகழ் புரிந்த இல் இலோர்க்கு இல்லை இகழ்வார்முன் 
ஏறு போல் பீடு நடை
@snapend

@snap[span-100 text-08 text-left]

> குடும்ப புகழை காக்க விரும்பும் மனைவி இல்லாதவருக்கு மக்கள் கூடிய சாலையில் தலைநிமிர்ந்து நடக்க முடியாது.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

```
மங்கலம் என்ப மனைமாட்சி மற்றதன்
நன்கலம் நன்மக்கட் பேறு		( குறள் - 60)

```

@snap[span-100 text-left text-04]
தொடரமைப்பு: மங்கலம் என்ப மனை மாட்சி 
அதன் நன் கலம் நன் மக்கள் பேறு.
@snapend

@snap[span-100 text-08 text-left]

> ஒருவனுக்கு நல்ல பண்புகள் உள்ள மனைவியே அழகு. அந்த அழகிற்கு ஏற்ற அணிகலன்கள் நல்லப் பிள்ளைகளை பெறுவதே.

@snapend

---?include=includes/VaazhkkaiThunainalam.md

@snap[span-100 text-08 text-left]

<div class="conclusion" >
இல்லறத்திற்கு உரிய பண்புகளாவன:

* விருந்தினரை உபசரித்தல்
* உறவினருடன் ஒற்றுமையாக இருத்தல்
* வறுமையில் இருப்பவருக்கு உதவுதல்
* நல்ல சமையல் திறன்
* கணவன் செய்யும் அறச்செயல்களுக்கு துணையாய் இருத்தல்

</div>

@snapend

---?include=includes/VaazhkkaiThunainalam.md
@snap[span-100 text-08 text-left]
<div class="conclusion" >

* தன் கடமைகளை தன்னிடம் இருப்பதை கொண்டு வருமானத்துக்குள் செய்யவேண்டும்

* அந்த பெண் மனவுறுதியோடு தன்னை தானே கற்பு நெறியில் காத்துக்கொண்டு தன்  கணவனையும், தன் குடும்பத்து புகழையும் காக்கவேண்டும்

* கணவனுக்கு உண்மையானவளாக கற்புநெறியில் வாழும் பெண்ணிற்கு இந்த இயற்கையே கட்டுப்படும் 

* இல்லறத்திற்கு அழகு வாழ்க்கைத்துணையின்  பண்புதான். அந்த அழகிற்கு அணிகலன் நல்ல பிள்ளைகள் 

* அப்படிப்பட்ட பண்புள்ள பெண்ணை துணையாக கொண்டவர் ஊரார் முன் தலைநிமிர்ந்து வாழலாம்   
</div>

> அன்பு தான் ஒவ்வொருவரிடமும் விருப்பமான ஆர்வத்தை உண்டாக்கும். அந்த ஆர்வமே பிறரிடம் நட்பை உருவாக்கும். 

@snapend

### அன்புடையவர் அடையும் சிறப்பு என்ன ?

---?include=includes/Anbudaimai.md

```
அன்புற் றமர்ந்த வழக்கென்ப வையகத்து
இன்புற்றார் எய்தும் சிறப்பு     ( குறள் - 75)
```

@snap[span-100 text-left text-04] 
**தொடரமைப்பு:**  அன்புற்று அமர்ந்த வழக்கு என்ப 
வையகத்து இன்புற்றார் எய்தும் சிறப்பு 

@snapend

@snap[span-100 text-08 text-left]

> அன்புடையராக இருந்து வாழுகின்ற வாழ்க்கையின் பயன் அவர்கள் இந்த உலகத்திலும் இன்பமாக வாழ்ந்து சொர்கத்திலும் இன்பமாக வாழலாம். 

@snapend

---?include=includes/Anbudaimai.md

### பகையையும் அன்பால் வெல்லமுடியுமா ?

---?include=includes/Anbudaimai.md

```
அறத்திற்கே அன்புசார் பென்ப அறியார்
மறத்திற்கும் அஃதே துணை      ( குறள் - 76)
```

@snap[span-100 text-left text-04] 

**தொடரமைப்பு:**  அன்பு சார்பு அறத்திற்கே என்ப அறியார்
மறத்திற்கும் அஃதே துணை.

@snapend

@snap[span-100 text-08 text-left]

> அன்பு துணையாவது அறத்திற்கே என்பர் அறியார், கோபத்தை நீக்குவதற்கும் அவ்வன்பே துணையாகும்.

@snapend

---?include=includes/Anbudaimai.md

### அன்பில்லாமல் குற்றங்கள் செய்தால் என்னாகும் ?

---?include=includes/Anbudaimai.md

```
என்பி லதனை வெயில்போலக் காயுமே
அன்பி லதனை அறம்     ( குறள் - 77)
```

@snap[span-100 text-left text-04] 

**தொடரமைப்பு:**  என்பு  இல் அதனை வெயில் போலக் காயுமே
அன்பு இல் அதனை அறம்.

@snapend

@snap[span-100 text-08 text-left]

> எலும்பு இல்லாத புழுவை வெயில் வருத்திக் கொல்லும். அதுபோல் அன்பு இல்லாதவரை அறம் வருத்திக் கொல்லும்.

@snapend

---?include=includes/Anbudaimai.md

### அன்பில்லாத வாழ்க்கை எப்படி இருக்கும்?

---?include=includes/Anbudaimai.md

```
அன்பகத் தில்லா உயிர்வாழ்க்கை வன்பாற்கண்
வற்றல் மரந்தளிர்த் தற்று        ( குறள் - 78)
```

@snap[span-100 text-left text-04] 

**தொடரமைப்பு:**  அகத்து அன்பு இல்லா உயிர்வாழ்க்கை வன் பால் கண்
வற்றல் மரம் தளிர்த் தற்று.

@snapend

@snap[span-100 text-08 text-left]

> மனதளவில் அன்பு இல்லாதவர்கள் வாழ்க்கை பாலை வனத்தில் பட்டமரம் தளிர்ப்பதைப் போன்றதாகும்.

@snapend

---?include=includes/Anbudaimai.md

### அன்பில்லாதவரிடம் இருக்கும் வசதிகள் பயன் தருமா?

---?include=includes/Anbudaimai.md

```
புறத்துறுப் பெல்லாம் எவன்செய்யும் யாக்கை
அகத்துறுப் பன்பி லவர்க்கு       ( குறள் - 79)
```

@snap[span-100 text-left text-04] 

**தொடரமைப்பு:** யாக்கை அகத்து  உறுப்பு அன்பு இலவர்க்கு  
புறத்து உறுப்பு எல்லாம் எவன் செய்யும்.

@snapend

@snap[span-100 text-08 text-left]

> மனைவியிடத்து  அன்பு இல்லாதவருக்கு , புறத்து வசதிகளாகிய வீடு, பணம், பணியாள் என்று எல்லாம் இருந்தும் அவை ஒரு பயனையும் தராது.

@snapend

### அன்பில்லாதவர் எப்படி பட்டவர்?

---?include=includes/Anbudaimai.md

```
அன்பின் வழிய துயிர்நிலை அஃதிலார்க்கு
என்புதோல் போர்த்த உடம்பு        ( குறள் - 80)
```

@snap[span-100 text-left text-04] 

**தொடரமைப்பு:**  அன்பின் வழியது உயிர்நிலை அஃது இலார்க்கு
உடம்பு என்பு தோல் போர்த்த. 

@snapend

@snap[span-100 text-08 text-left]

> அன்பின் வழியில் இயங்கும் உடம்பே உயிர் உள்ள உடம்பு. அன்பு இல்லாதவருக்கு உள்ள உடம்பு எலும்பைத் தோல் போர்த்த வெற்றுடம்பே ஆகும்.

@snapend

```
இருந்தோம்பி இல்வாழ்வ தெல்லாம் விருந்தோம்பி
```

### விருந்து உபசரிக்கும் முறை என்ன?

---?include=includes/Virundhombal.md

```
விருந்து புறத்ததாத் தானுண்டல் சாவா
மருந்தெனினும் வேண்டற்பாற் றன்று		( குறள் - 82)

```

@snap[span-100 text-08 text-left]

> விருந்தினர் வீட்டிற்கு வெளியே இருக்கத் தான் மட்டும் தனித்து உண்பது சாவைத் தடுக்கும் மருந்தே என்றாலும் விரும்பத் தக்கது அன்று.

@snapend

---?include=includes/Virundhombal.md

### விருந்து எனும் அறம் செய்தால் வறுமை வருமா?

---?include=includes/Virundhombal.md

```
வருவிருந்து வைகலும் ஓம்புவான் வாழ்க்கை
பருவந்து பாழ்படுதல் இன்று		( குறள் - 83)

```

@snap[span-100 text-08 text-left]

> தன்னை நோக்கி வரும் விருந்தினரை நாள்தோறும் போற்றுகின்றவனுடைய வாழ்க்கை வறுமைப் பட்டுக் கெட்டுப் போவது இல்லை.

@snapend

---?include=includes/Virundhombal.md

### செலவு செய்து விருந்து வைத்தாலும் வறுமை வராதா ? அது எப்படி?

---?include=includes/Virundhombal.md

```
அகனமர்ந்து செய்யாள் உறையும் முகனமர்ந்து
நல்விருந் தோம்புவான் இல்		( குறள் - 84)

```
@snap[span-100 text-08 text-left]

> இனிய முகத்தோடு தக்க விருந்தினரைப் பேணுபவரின் வீட்டில் மஹாலட்சுமி மனம் மகிழ்ந்து குடி இருப்பாள்.

@snapend

### விருந்தினரை உபசரித்தால் கிடைக்கும் நன்மைகள் என்ன?

---?include=includes/Virundhombal.md

```
வித்தும் இடல்வேண்டும் கொல்லோ விருந்தோம்பி
மிச்சில் மிசைவான் புலம்		( குறள் - 85)

```

@snap[span-100 text-08 text-left]

> விருந்தினர் முதலில் உண்ண மிஞ்சியவற்றை உண்பவரின் நிலத்தில் விதை விதைக்கவும் வேண்டுமோ? தேவை இல்லை செல்வம் தானே பெருகும்.

@snapend

---?include=includes/Virundhombal.md

### நாள்தோறும் விருந்தினரை உபசரிப்பவர்களின் சிறப்பு என்ன?

---?include=includes/Virundhombal.md

```
செல்விருந் தோம்பி வருவிருந்து பார்த்திருப்பான்
நல்விருந்து வானத் தவர்க்கு		( குறள் - 86)

```

@snap[span-100 text-08 text-left]

> வந்த விருந்தினரை உபசரித்து, இனி வரப் போகும் விருந்தினரை எதிர் பார்த்திருப்பவன் வானுலகத்தில் உள்ள தேவர்க்கு நல்ல விருந்தினனாவான்.

@snapend

---?include=includes/Virundhombal.md

### விருந்தோம்பலின் பயன் என்ன?

---?include=includes/Virundhombal.md

```
இனைத்துணைத் தென்பதொன் றில்லை விருந்தின்
துணைத்துணை வேள்விப் பயன்		( குறள் - 87)

```

@snap[span-100 text-08 text-left]

> விருந்தினரை உபசரித்தல் என்பது ஒரு யாகமே. அதைச் செய்தால் வரும் நன்மை இவ்வளவு என்று அளவிட முடியாது. வரும் விருந்தினரின் தகுதி அளவு தான் நன்மையின் அளவாகும்.

@snapend

---?include=includes/Virundhombal.md

### விருந்து செய்யவில்லை என்றால் என்ன ஆகும்?

---?include=includes/Virundhombal.md

```
பரிந்தோம்பிப் பற்றற்றேம் என்பர் விருந்தோம்பி
வேள்வி தலைப்படா தார்		( குறள் - 88)

```

@snap[span-100 text-08 text-left]

> செல்வத்தைச் சிரமப்பட்டுக் காத்தும் அதனை இழக்கும்ப் போது இப்போது எந்தத் துணையும் இல்லாதவராய் போனோமே என்று வருந்துவர், விருந்தினரைப் பேணி அந்த யாகத்தின் பயனைப் பெறும் பேறு அற்றவர்.

@snapend

---?include=includes/Virundhombal.md

### செல்வ நிலையிலும் உள்ள வறுமை எது ?

---?include=includes/Virundhombal.md

```
உடைமையுள் இன்மை விருந்தோம்பல் ஓம்பா
மடமை மடவார்கண் உண்டு		( குறள் - 89)

```

@snap[span-100 text-08 text-left]

> செல்வநிலையில் உள்ள வறுமை என்பது விருந்தோம்பலைப் போற்றாத அறியாமையாகும். அது அறிவிலிகளிடத்தில் இருக்கும்.

@snapend

---?include=includes/Virundhombal.md

### ஏன் இனிய முகத்துடன் விருந்துக் கொடுக்க வேண்டும்.

---?include=includes/Virundhombal.md

```
மோப்பக் குழையும் அனிச்சம் முகந்திரிந்து
நோக்கக் குழையும் விருந்து		( குறள் - 90)

```

@snap[span-100 text-08 text-left]

> அனிச்சப்பூ மோந்தவுடன் வாடிவிடும். அதுபோல் விருந்தினர் நம் முகம் வேறுப்பட்டுப் பார்த்த அளவில் வாடிவிடுவர்.

கொடுக்க கொடுக்க செல்வம் பெருகும்.

</div>

###### உங்களுக்கு இந்த அதிகாரத்துலே எந்த குறள் பிடிச்சிருக்குனு கமெண்ட்-ல சொல்லுங்க.

---?include=includes/IniyavaiKooral.md

### இனியவை கூறல் : இனிமையான சொற்களை பேசுவது 

---?include=includes/IniyavaiKooral.md

### ஒரு சொல் வெல்லும், ஒரு சொல் கொல்லும்

---?include=includes/IniyavaiKooral.md

@img[width=250, height=250, shadow](assets/img/bharathiyar1.png)

```
மனதில் உறுதி வேண்டும். வாக்கினிலே இனிமை வேண்டும் ( பாரதியார் )

```
---?include=includes/IniyavaiKooral.md

@img[width=250, height=250, shadow](assets/img/vallalar1.png)

```
உள் ஒன்று வைத்துப் புறம் ஒன்று பேசுவார் உறவு கலவாமை வேண்டும் ( வள்ளலார் )
```

---?include=includes/IniyavaiKooral.md
### இன்சொல் என்பது எப்படி இருக்க வேண்டும் ?

---?include=includes/IniyavaiKooral.md

```
இன்சொலால் ஈரம் அளைஇப் படிறிலவாம்
செம்பொருள் கண்டார்வாய்ச் சொல்		( குறள் - 91)

```

@snap[span-100 text-08 text-left]

> இன்சொல் என்பது அன்பு கலந்ததாகவும், வஞ்சனை அற்றதாகவும், உண்மை உடையதாகவும் இருக்க வேண்டும்.

@snapend

---?include=includes/IniyavaiKooral.md

### முகமலர்ச்சி, இன்சொல், ஈதல் இவற்றுள் எவை சிறந்தது ?

---?include=includes/IniyavaiKooral.md

```
அகன்அமர்ந் தீதலின் நன்றே முகனமர்ந்து
இன்சொலன் ஆகப் பெறின்		( குறள் - 92)

```

@snap[span-100 text-08 text-left]

> முகம் மலர்ந்து இன்சொல் உடையவனாக இருக்கப் பெற்றால், மனம் மகிழ்ந்து பொருள் கொடுக்கும் ஈகையை விட அது சிறந்தது.

@snapend

---?include=includes/IniyavaiKooral.md

### எது அறம் ?

---?include=includes/IniyavaiKooral.md

```
முகத்தான் அமர்ந்தின்து நோக்கி அகத்தானாம்
இன்சொ லினதே அறம்		( குறள் - 93)

```

@snap[span-100 text-08 text-left]

> பிறரைப் பார்க்கும் பொழுது முகத்தால் விரும்பி, இனிமையாகப் பார்த்து, உள்ளத்துள் இருந்து வரும் இனியச் சொற்களைப் பேசுவதே அறம்.

@snapend

---?include=includes/IniyavaiKooral.md

### இன்சொல் பேசுவதால் வரும் நன்மைகள் என்ன ?

---?include=includes/IniyavaiKooral.md
```
துன்புறூஉம் துவ்வாமை இல்லாகும் யார்மாட்டும்
இன்புறூஉம் இன்சொ லவர்க்கு		( குறள் - 94)

```

@snap[span-100 text-08 text-left]

> எல்லோரிடமும் இன்பம் தரும் இனியச் சொற்களை பேசுபவர்க்கு துன்பம் தரும் வறுமை வராது.

@snapend

---?include=includes/IniyavaiKooral.md

### எவை உண்மையான அழகு ?

---?include=includes/IniyavaiKooral.md
```
பணிவுடையன் இன்சொலன் ஆதல் ஒருவற்கு
அணியல்ல மற்றுப் பிற		( குறள் - 95)

```

@snap[span-100 text-08 text-left]

> ஒருவர்க்கு அணி என்பது வணங்கக் கூடியவரை வணங்குவதும், எல்லோரிடமும் இனிய சொற்களை கூறுவதும் ஆகும். இவை தவிர மற்றவை அணி அல்ல.

@snapend

---?include=includes/IniyavaiKooral.md

### உலகில் அறம் பெருக என்ன செய்ய வேண்டும் ?

---?include=includes/IniyavaiKooral.md

```
அல்லவை தேய அறம்பெருகும் நல்லவை
நாடி இனிய சொலின்		( குறள் - 96)

```

@snap[span-100 text-08 text-left]

> பிறர்க்கு நன்மை தரும் இனிய சொற்களை மனதால் தேர்ந்தெடுத்து சொன்னால், பாவங்கள் குறைய அறம் வளர்ந்து பெருகும்.

@snapend

---?include=includes/IniyavaiKooral.md

### பிறர்க்குப் பயன் தரும் நல்ல சொற்களை பேசுவதன் நன்மை என்ன ?

---?include=includes/IniyavaiKooral.md

```
நயன்ஈன்று நன்றி பயக்கும் பயன்ஈன்று
பண்பின் தலைப்பிரியாச் சொல்		( குறள் - 97)

```

@snap[span-100 text-08 text-left]

> பிறர்க்கு நன்மையான பயனைத் தந்து நல்லப் பண்பிலிருந்து நீங்காத சொற்கள், உலகத்தாரோடு ஒற்றுமையையும், நன்மையையும் கொடுக்கும்.

@snapend

---?include=includes/IniyavaiKooral.md

### பிறரை இழிவுபடுத்தாத இனிய சொற்களை பேசுவதன் நன்மை என்ன ?

---?include=includes/IniyavaiKooral.md

```
சிறுமையுள் நீங்கிய இன்சொல் மறுமையும்
இம்மையும் இன்பம் தரும்		( குறள் - 98)

```

@snap[span-100 text-08 text-left]

> பிறர்க்கு மனத்துன்பம் தராத இனிய சொல், ஒருவனுக்கு இருமையிலும் இன்பத்தை தரும்.

@snapend

---?include=includes/IniyavaiKooral.md

### கடுஞ்சொல் பேசுபவரிடம் திருவள்ளுவர் கேட்கும் கேள்வி என்ன ?

---?include=includes/IniyavaiKooral.md

```
இன்சொல் இனிதீன்றல் காண்பான் எவன்கொலோ
வன்சொல் வழங்கு வது		( குறள் - 99)

```

@snap[span-100 text-08 text-left]

> பிறர் கூறும் இன்சொல் தனக்கு இன்பம் பயத்தலை அனுபவித்து அறிகின்றவன், பிறர் இடத்து வன்சொல்லை சொல்வது என்ன பயன் கருதி ?

@snapend

---?include=includes/IniyavaiKooral.md

### கடுஞ்சொல்  பேசுவது எப்படிப் பட்டது ?

---?include=includes/IniyavaiKooral.md

```
இனிய உளவாக இன்னாத கூறல்
கனிஇருப்பக் காய்கவர்ந் தற்று		( குறள் - 100)

```

@snap[span-100 text-08 text-left]

> இனிய சொற்கள் இருக்கும் பொழுது அவற்றை விட்டு கடுமையான சொற்களை கூறுதல், இனிய கனிகள் இருக்க அதை உண்ணாமல் காய்களை உண்பதாகும்.

@snapend

---?include=includes/IniyavaiKooral.md

@snap[span-100 text-08 text-left]
<div class="conclusion" >
கடுமையான சொற்களை தவிர்த்து, இனிமையான சொற்களை பேச வேண்டும்.
</div>

<div class="conclusion" >
அப்படி பேசுவதால், குடும்பத்திலும் சமூகத்திலும் நல்ல சுமுகமான உறவு ஏற்படும்.
</div>

@snapend

---?include=includes/IniyavaiKooral.md

@snap[span-100 text-08 text-left]
<div class="conclusion" >
நல்ல விஷயங்களை சொல்லும் போது கூட, இனிமையான வார்த்தைகளை பயன்படுத்த வேண்டும்.
</div>

<div class="conclusion" >
உதாரணமா: நல்லா படிக்கலனா fail ஆயிடுவ -னு சொல்லாம 
ஒழுங்கா படிச்சா pass ஆகலாம்ல -னு சொல்லலாம்.
</div>

@snapend

### செய்ந்நன்றி அறிதல் : பிறர் நமக்கு செய்த உதவியை மறக்க கூடாது.

### பிறர் முன் ஒரு உதவி செய்யாமல் நாம் அவருக்கு செய்யும் உதவியின் சிறப்பு என்ன?

---?include=includes/SeiNandriArithal.md

```
செய்யாமல் செய்த உதவிக்கு வையகமும்
```

### சரியான சமயத்தில் செய்யும் உதவி எவ்வளவு பெரியது ?

```
காலத்தி னாற்செய்த நன்றி சிறிதெனினும்

```

### பயன் எதிர்பார்க்காமல் செய்யும் உதவி எவ்வளவு பெரியது?

---?include=includes/SeiNandriArithal.md

```
பயன்தூக்கார் செய்த உதவி நயன்தூக்கின்
நன்மை கடலின் பெரிது		( குறள் - 103)

```

@snap[span-100 text-08 text-left]

> எவ்விதப் பயனையும் எதிர்பார்க்காமல் ஒருவன் செய்த உதவியின் அன்பை மட்டும் ஆராய்ந்தால், அதன் நன்மை கடலை விடப் பெரியதாகும்.

@snapend

---?include=includes/SeiNandriArithal.md

### திருவள்ளுவர், உதவி பெறுபவர்கள் பற்றி என்ன சொல்கிறார்?

---?include=includes/SeiNandriArithal.md

```
தினைத்துணை நன்றி செயினும் பனைத்துணையாக்
கொள்வர் பயன்தெரி வார்		( குறள் - 104)

```

@snap[span-100 text-08 text-left]

> செய்யப்பட்ட உதவி சிறிதே என்றாலும் அதனால் பயன் பெறும் பண்புடையவர்கள் அதனை மிகப் பெரிய உதவியாகக் கருதுவர்.

@snapend

### உதவியின் மதிப்பு எதை பொறுத்தது ?

---?include=includes/SeiNandriArithal.md

```
உதவி வரைத்தன் றுதவி உதவி
செயப்பட்டார் சால்பின் வரைத்து		( குறள் - 105)

```

@snap[span-100 text-08 text-left]

> ஒருவர் நமக்கு செய்த உதவிக்குத் திரும்ப நாம் செய்வது அவர் செய்த உதவியின் அளவை பார்த்து அன்று. அந்த உதவியைப் பெற்றவரின் பண்பை பொறுத்தது.

@snapend

---?include=includes/SeiNandriArithal.md

### நாம் நம் வாழ்வில் யாரை மறக்கவே கூடாது ?

---?include=includes/SeiNandriArithal.md

```
மறவற்க மாசற்றார் கேண்மை துறவற்க
துன்பத்துள் துப்பாயார் நட்பு		( குறள் - 106)

```

@snap[span-100 text-08 text-left]

> உன் துன்பத்தில் உனக்கு துணையாய் நின்றவரின் நட்பை விடாதே; அறிவு ஒழுக்கங்களில் குற்றம் இல்லாதவரின் நட்பை மறந்து விடாதே.

@snapend

### நம் வாழ்வில் யாரை எப்போதும் நினைவில் வைத்து கொள்ள வேண்டும்?

---?include=includes/SeiNandriArithal.md

```
எழுமை எழுபிறப்பும் உள்ளுவர் தங்கண்
விழுமந் துடைத்தவர் நட்பு		( குறள் - 107)

```

@snap[span-100 text-08 text-left]

> தம் துன்பத்தை போக்கியவரின் நட்பை ஏழேழு பிறப்பிலும் நல்லவர் எண்ணுவர்.

@snapend

---?include=includes/SeiNandriArithal.md

### நாம் நம் வாழ்வில் நிம்மதியாகவும் மகிழ்ச்சியுடனும் வாழ்வதற்கு என்ன செய்ய வேண்டும் ?

---?include=includes/SeiNandriArithal.md

```
நன்றி மறப்பது நன்றன்று நன்றல்லது
அன்றே மறப்பது நன்று		( குறள் - 108)

```

@snap[span-100 text-08 text-left]

> ஒருவர் நமக்கு செய்த உதவியை மறப்பது நல்லதல்ல. அவர் செய்த தீமையை அப்போதே மறப்பது நன்று.

@snapend

---?include=includes/SeiNandriArithal.md

### நமக்கு தீமை செய்தவர்களை மன்னிப்பதற்க்கு என்ன வழி ?

---?include=includes/SeiNandriArithal.md

```
கொன்றன்ன இன்னா செயினும் அவர்செய்த
ஒன்றுநன் றுள்ளக் கெடும்		( குறள் - 109)

```

@snap[span-100 text-08 text-left]

> நமக்கு முன் ஒரு நன்மை செய்தவர், நம்மை கொல்வதற்கு சமமான தீமை செய்தாலும், அவர் முன்பு செய்த ஒப்பற்ற நன்மையை நினைத்த அளவில் அத்தீமை மறையும்.

@snapend

---?include=includes/SeiNandriArithal.md

### எந்த பாவத்திற்கு மன்னிப்பு கிடையாது?

---?include=includes/SeiNandriArithal.md

```
எந்நன்றி கொன்றார்க்கும் உய்வுண்டாம் உய்வில்லை
செய்ந்நன்றி கொன்ற மகற்கு		( குறள் - 110)

```

@snap[span-100 text-08 text-left]

> எந்த அறத்தை அழித்தவருக்கும் பாவங்களை கழுவ வழி உண்டாகும். ஆனால் ஒருவர் செய்த உதவியை மறந்து அழித்தவருக்கு உருப்பட வழியே இல்லை.

@snapend

<div class="conclusion" >

நமக்கு உதவி செய்தவர்களை நாம் நன்றியோடு நினைக்க வேண்டும். நாமும் உதவிகள் செய்ய வேண்டும். 

</div>
@snapend

---?include=includes/NaduvuNilaimai.md

### இந்த அதிகாரத்தை எப்படி நம்ம வாழ்க்கையில பயன்ப்படுத்துறது?

</div> 


---?include=includes/NaduvuNilaimai.md

### நடுவு நிலைமை என்பது எப்படி இருக்க வேண்டும் ?

---?include=includes/NaduvuNilaimai.md

```
தகுதி யெனவொன்று நன்றே பகுதியாற்
பாற்பட் டொழுகப் பெறின்		( குறள் - 111)

```

@snap[span-100 text-08 text-left]

> பகைவர், நண்பர், புதியவர் என்னும் பிரிவினர்தோறும் நீதி தவறாது பின்பற்றப்படுமானால், நடுவு நிலைமை என்று சொல்லப்படும் ஓர் அறமே நன்று.

@snapend

---?include=includes/NaduvuNilaimai.md

### நாம் கஷ்டப்பட்டு சேர்த்த பணத்தை இழந்திடாமல் இருக்க என்ன செய்ய வேண்டும் ?

---?include=includes/NaduvuNilaimai.md

```
செப்பம் உடையவன் ஆக்கஞ் சிதைவின்றி
எச்சத்திற் கேமாப் புடைத்து		( குறள் - 112)

```

@snap[span-100 text-08 text-left]

> நீதியை உடையவனின் செல்வம் அழியாமல் அவன் வழியில் உள்ளவர்க்கும் உறுதியான நன்மை தரும்.

@snapend

---?include=includes/NaduvuNilaimai.md

### நடுவு நிலைமை தவறினால் நன்மை கிடைக்கும் என்ற சூழ்நிலையில் என்ன செய்ய வேண்டும்?

---?include=includes/NaduvuNilaimai.md

```
நன்றே தரினும் நடுவிகந்தாம் ஆக்கத்தை
அன்றே யொழிய விடல்		( குறள் - 113)

```

@snap[span-100 text-08 text-left]

> நன்மை தருவதாக இருந்தாலும் நீதியை விட்டு விலகுவதால் வரும் லாபத்தை அப்பொழுதே விட்டு விட வேண்டும்.

@snapend

---?include=includes/NaduvuNilaimai.md

### நாம் நடுவுநிலைமை தவறி நடந்தால் யாருக்கு தெரியப்போகிறது?

---?include=includes/NaduvuNilaimai.md

```
தக்கார் தகவிலர் என்ப தவரவர் 
எச்சத்தாற் காணப் படும்		( குறள் - 114)

```

@snap[span-100 text-08 text-left]

> நடுவு நிலைமை உடையவர், நடுவு நிலைமை இல்லாதவர் என்பது அவரவர்க்குப் பின் எஞ்சி நிற்கும் புகழாலும் பழியாலும் காணப்படும்.


@snapend

### எல்லா நேரமும் நேர்மையாகவே இருக்க முடியுமா? ஆதாயத்திற்காக மனம் மாறுமே ?

@snapend


> தன் நெஞ்சம் நீதி தவறி அநீதி செய்ய எண்ணினால், நான் கெடப்போகிறேன் என்று ஒருவன் அறிய வேண்டும்.

### நேர்மையாக இருந்தும் வறுமையில் வருந்துபவர்களுக்காக திருவள்ளுவர் என்ன சொல்கிறார் ?

@snapend

### ஏன் அடக்கத்துடன் வாழ வேண்டும்?

---?include=includes/Adakamudaimai.md

```
அடக்கம் அமரருள் உய்க்கும் அடங்காமை
ஆரிருள் உய்த்து விடும்		( குறள் - 121)

```

@snap[span-100 text-08 text-left]

> அடக்கம் ஒருவனை உயர்த்தித் தேவருள் சேர்க்கும். அடங்காமை வாழ்வையே இருளாக்கி விடும்.

@snapend

---?include=includes/Adakamudaimai.md

### நாம் நம் வாழ்வில் பொருளாக எதை காக்க வேண்டும்?

---?include=includes/Adakamudaimai.md

```
காக்க பொருளா அடக்கத்தை ஆக்கம்
அதனினூஉங் கில்லை உயிர்க்கு		( குறள் - 122)

```

@snap[span-100 text-08 text-left]

> அடக்கத்தை உறுதிப் பொருளாகக் கொண்டு போற்றிக் காக்க வேண்டும். அந்த அடக்கத்தைவிட மேம்பட்ட ஆக்கம் உயிர்க்கு இல்லை.

@snapend

---?include=includes/Adakamudaimai.md

### அடக்கமாக இருந்தால் வாழ்வில் எப்படி பிரபலம் ஆவது ?

---?include=includes/Adakamudaimai.md

```
செறிவறிந்து சீர்மை பயக்கும் அறிவறிந்
தாற்றின் அடங்கப் பெறின்		( குறள் - 123)

```

@snap[span-100 text-08 text-left]

> அறிய வேண்டியவற்றை அறிந்து, நல்வழியில் அடங்கி ஒழுகப்பெற்றால், அந்த அடக்கம் நல்லோரால் அறியப்பட்டு மேன்மை பயக்கும்.

@snapend

---?include=includes/Adakamudaimai.md

### அடக்கத்துடன் இருப்பவர்கள் எவ்வளவு பெரியவர்கள் ?

---?include=includes/Adakamudaimai.md

```
நிலையின் திரியா தடங்கியான் தோற்றம்
மலையினும் மாணப் பெரிது		( குறள் - 124)

```

@snap[span-100 text-08 text-left]

> நேர்மையான வழியை விட்டு விலகாது அடக்கத்துடன் வாழ்பவருடைய உயர்வு, மலையின் உயர்வை விட மிகவும் பெரியது.

@snapend

---?include=includes/Adakamudaimai.md

### செல்வம் இருக்கும் போது கூடவே கர்வம் வருவது இயல்புதானே ?

---?include=includes/Adakamudaimai.md

```
எல்லார்க்கும் நன்றாம் பணிதல் அவருள்ளும்
செல்வர்க்கே செல்வம் தகைத்து		( குறள் - 125)

```

### அடக்கவேண்டியவற்றுள் மிக முக்கியமாக அடக்க வேண்டியது எதை?

```
யாகாவா ராயினும் நாகாக்க காவாக்காற்

```

@snap[span-100 text-08 text-left]

> எதைக் காக்க முடியாதவரானாலும் நா ஒன்றையாவது காத்துக் கொள்ள வேண்டும். முடியாது போனால் சொல்குற்றத்தில் சிக்கித் துன்பப்படுவர்.

@snapend

### எல்லா நேரமும் யோசித்து பேச முடியுமா? தவறி ஒரு வார்த்தை தீச்சொல் பேசுவது குற்றமா?

---?include=includes/Adakamudaimai.md

```
ஒன்றானுந் தீச்சொல் பொருட்பயன் உண்டாயின்
நன்றாகா தாகி விடும்		( குறள் - 128)

```

@snap[span-100 text-08 text-left]

> ஒரு குடம் பாலில் ஒரு துளி நஞ்சு போல், பேசும் சொற்களில் ஒரு சொல் தீய சொல்லாக இருந்து துன்பம் விளைவிக்குமானாலும், அந்தப் பேச்சில் உள்ள நல்ல சொற்கள் அனைத்தும் தீயதாகிவிடும்.

@snapend

```
நாவினாற் சுட்ட வடு		( குறள் - 129)

```

@snap[span-100 text-08 text-left]

> தீயினால் சுட்ட புண்கூட ஆறி விடும். ஆனால் வெறுப்புக் கொண்டு திட்டிய சொற்கள் விளைத்த துன்பம் ஆறவே ஆறாது.

@snapend


@snap[span-100 text-08 text-left]

> ஒருவர் நமக்கு தீங்கு செய்கிறார். அவரை அடக்கமுடியும் என்றாலும் பொறுத்துக்கொள்வது நல்லது. அதை விட அதனை மறந்து விடுவது அதனினும் நல்லது.

@snapend

---?include=includes/Poraiyudaimai.md

### தீமையை பொறுத்து போவது பலவீனம் என்று ஆகாதா ?

---?include=includes/Poraiyudaimai.md

```
இன்மையுள் இன்மை விருந்தொரால் வன்மையுள்
வன்மை மடவார்ப் பொறை		( குறள் - 153)

```

@snap[span-100 text-08 text-left]

> வறுமையில் வறுமை வந்த விருந்தினரை உபசரிக்காதது. வலிமையுள் வலிமை என்பது அறிவிலார் தீங்கு செய்தலை பொறுத்தலாகும்.

@snapend

---?include=includes/Poraiyudaimai.md

### உயர்வான மனிதர்களாக வாழ எப்படி நடந்துகொள்ள வேண்டும்?

---?include=includes/Poraiyudaimai.md

```
நிறையுடைமை நீங்காமை வேண்டின் பொறையுடைமை
போற்றி ஒழுகப் படும்		( குறள் - 154)

```

@snap[span-100 text-08 text-left]

> உயர்ந்த மனிதன் என்ற தன்மை நம்மை விட்டு நீங்காமல் இருக்க வேண்டுமானால், பொறுமையை போற்றி காக்க வேண்டும்.

@snapend

---?include=includes/Poraiyudaimai.md

### நாம் ஏன் உயர்ந்த மனிதர்களாக வாழ வேண்டும்?

---?include=includes/Poraiyudaimai.md

```
ஒறுத்தாரை ஒன்றாக வையாரே வைப்பர்
பொறுத்தாரைப் பொன்போல் பொதிந்து		( குறள் - 155)

```

@snap[span-100 text-08 text-left]

> தனக்குத் தீமை செய்தவரைப் பொறுக்காமல் தண்டிப்பவரைப் பெரியோர் ஒரு பொருட்டாக மதிக்க மாட்டார். பொறுத்தவரைப் பொன் போல் மனதுக்குள் வைத்து மதிப்பர்.

@snapend

---?include=includes/Poraiyudaimai.md

### தீமை செய்தவரை தண்டித்தால் தானே நான் இன்பமாக இருக்க முடியும்?

---?include=includes/Poraiyudaimai.md

```
ஒறுத்தார்க்கு ஒருநாளை இன்பம் பொறுத்தார்க்குப்
பொன்றும் துணையும் புகழ்		( குறள் - 156)

```

@snap[span-100 text-08 text-left]

> தமக்குத் தீங்கு செய்தவரைத் தண்டித்தவர்க்குத் தண்டித்த அன்று மட்டுமே இன்பம். பொறுத்துக்  கொண்டோர்க்கோ உலகம் உள்ளவரை புகழ் இருக்கும். 

@snapend

---?include=includes/Poraiyudaimai.md

### நமக்கு தீமை செய்தவரை நாம் ஏன் பொறுத்து போக வேண்டும்?

---?include=includes/Poraiyudaimai.md

```
திறன்அல்ல தற்பிறர் செய்யினும் நோநொந்து 
அறன்அல்ல செய்யாமை நன்று		( குறள் - 157)

```

@snap[span-100 text-08 text-left]

> கொடியவற்றைப் பிறர் தனக்குச் செய்தாலும், பதிலுக்குத் தானும் செய்தால் அவர் வருந்துவாரே என வருந்தி, அறம் அல்லாதவற்றைச் செய்யாமல் இருப்பது நல்லது.

@snapend

---?include=includes/Poraiyudaimai.md

### பொறுத்து போனால் வாழ்வில் வெற்றி கிடைக்குமா?

---?include=includes/Poraiyudaimai.md

```
மிகுதியான் மிக்கவை செய்தாரைத் தாம்தம் 
தகுதியான் வென்று விடல்		( குறள் - 158)

```

@snap[span-100 text-08 text-left]

> ஆணவத்தால் தீமை செய்தவரைப் பொறுமையால் வென்று விடுக.

@snapend

---?include=includes/Poraiyudaimai.md

### பொறுமையாய் இருப்பவர்கள் எவ்வளவு உயர்ந்தவர்கள் ?

---?include=includes/Poraiyudaimai.md

```
துறந்தாரின் தூய்மை உடையர் இறந்தார்வாய்
இன்னாச்சொல் நோற்கிற் பவர்		( குறள் - 159)

```

@snap[span-100 text-08 text-left]

> வரம்பு கடந்து நடப்பவரின் வாயில் பிறக்கும் கொடுஞ்சொற்களைப் பொறுத்துக் கொள்பவர், இல்வாழ்க்கையில் வாழ்ந்தாலும் துறவியை விட தூயவர்.

@snapend

---?include=includes/Poraiyudaimai.md

### தீய சொற்களை பொறுத்து போவது எவ்வளவு பெரியது?

---?include=includes/Poraiyudaimai.md

```
உண்ணாது நோற்பார் பெரியர் பிறர்சொல்லும்
இன்னாச்சொல் நோற்பாரின் பின்		( குறள் - 160)

```

@snap[span-100 text-08 text-left]

> உண்ணாமல் விரதம் இருப்பவர்கள் துறவிகள். மற்றவர்கள் சொல்லும் தீயசொற்களைப் பொறுத்துக்கொள்பவர்கள்  அந்த துறவிகளை விட பெரியவர்கள். 

@snapend

ஏன் பொறாமை குணம் இருக்க கூடாது ?

பொறாமை குணம் மனக்குற்றங்களில் ஒன்று. 

---?include=includes/Alukkaramai.md

துரியோதனன் 

---?include=includes/Alukkaramai.md

இந்த அதிகாரம் யாருக்கெல்லாம் பயன்படும் ?

---?include=includes/Alukkaramai.md

அடுத்தவர்கள் நம் குற்றங்களை சுட்டி காட்டுவதற்கு முன் நாம் நம் குற்றங்களை நீக்கி மனத்தூய்மையோடு வாழ நினைக்கும் அனைவருக்கும். 

---?include=includes/Alukkaramai.md

நம்மிடம் பொறாமை குணம் இருக்கின்றதா என்று எப்படி தெரிந்து கொள்வது?

---?include=includes/Alukkaramai.md

1 . நம்மை அடுத்தவருடம் ஒப்பிட்டு பார்ப்பது. 

2 . பிறரின் வெற்றி அவர்களின் அதிஷ்டத்தால் கிடைத்தாக நினைப்பது. 

3 . பிறரின் வெற்றி கண்டு எரிச்சல். 

---?include=includes/Alukkaramai.md

பொறாமை குணத்தை எப்படி நீக்குறது?

அதை தெரிந்து கொள்ள இந்த காணொளியை இறுதி வரை பாருங்க. 

---?include=includes/Alukkaramai.md

### நாம் நம் வாழ்வில் மிக முக்கியமாக கடைபிடிக்க வேண்டிய ஒழுக்கம் எது ?

---?include=includes/Alukkaramai.md

```
ஒழுக்காறாக் கொள்க ஒருவன்தன் நெஞ்சத்து 
அழுக்காறு இலாத இயல்பு		( குறள் - 161)

```

@snap[span-100 text-08 text-left]

> உள்ளத்துள் பொறாமை இல்லாமல் வாழும் குணத்தை, ஒருவன் தனக்கு உரிய ஒழுக்கமாகக் கொள்க.

@snapend

---?include=includes/Alukkaramai.md

### ஏன் பொறாமை குணம் இல்லாமல் வாழ வேண்டும் ?

---?include=includes/Alukkaramai.md

```
விழுப்பேற்றின் அஃது ஒப்பது இல்லை யார் மாட்டும்
அழுக்காற்றின் அன்மை பெறின்		( குறள் - 162)

```

@snap[span-100 text-08 text-left]

> யாரிடமும் பொறாமை கொள்ளாத பண்பு ஒருவர்க்கு வாய்க்கப் பெறுமேயானால் அதற்கு மேலான பேறு அவருக்கு வேறு ஏதும் இல்லை.

@snapend

---?include=includes/Alukkaramai.md

### பொறாமை குணம் இருப்பவர்கள் எப்படி பட்டவர்கள்?

---?include=includes/Alukkaramai.md

```
அறன் ஆக்கம் வேண்டாதான் என்பான் பிறனாக்கம்
பேணாது அழுக்கறுப் பான்		( குறள் - 163)

```

@snap[span-100 text-08 text-left]

> தனக்கு அறமும் ஆக்கமும் விரும்பாதவன் என்று கருதத் தக்கவனே, பிறனுடைய ஆக்கத்தைக் கண்டு மகிழாமல் அதற்காகப் பொறாமைப்படுவான்.

@snapend

---?include=includes/Alukkaramai.md

### பொறாமை குணம் இருந்தால் என்ன ஆகும் ?

---?include=includes/Alukkaramai.md

```
அழுக்காற்றின் அல்லவை செய்யார் இழுக்காற்றின்
ஏதம் படுபாக்கு அறிந்து		( குறள் - 164)

```

@snap[span-100 text-08 text-left]

> தீய வழியில் சென்றால் துன்பம் ஏற்படுமென்பதை அறிந்தவர்கள் பொறாமையினால் தீச்செயல்களில் ஈடுபடமாட்டார்கள்.

@snapend

---?include=includes/Alukkaramai.md

### பொறாமை குணம் இருந்தால் துன்பம் யாரால் வரும்?

---?include=includes/Alukkaramai.md

```
அழுக்காறு உடையார்க்கு அதுசாலும் ஒன்னார்
வழுக்கியும்  கேடுஈன் பது		( குறள் - 165)

```

@snap[span-100 text-08 text-left]

> பொறாமை உடையவருக்குத் தீமை தரப் பகைவர் வேண்டியதில்லை. பொறாமையே போதும்.

@snapend

---?include=includes/Alukkaramai.md

### எதற்காக பொறாமைப் படுவது மிக கீழ்தரமானது?

---?include=includes/Alukkaramai.md

```
கொடுப்பது அழுக்கறுப்பான் சுற்றம் உடுப்பதூஉம்
உண்பதூஉம் இன்றிக் கெடும்		( குறள் - 166)

```

@snap[span-100 text-08 text-left]

> உதவியாக ஒருவருக்குக் கொடுக்கப்படுவதைப் பார்த்துப் பொறாமை கொண்டால் அந்தத் தீய குணம், அவனை மட்டுமின்றி அவனைச் சார்த்திருப்போரையும் உணவுக்கும், உடைக்கும்கூட வழியில்லாமல் ஆக்கிவிடும்.

@snapend

---?include=includes/Alukkaramai.md

### நிறைய நல்ல குணங்கள்  உள்ள நான் சில சமயங்களில் அடுத்தவரை பார்த்து பொறாமைப் பட்டால் என்ன ஆகும்?

---?include=includes/Alukkaramai.md

```
அவ்வித்து அழுக்காறு உடையானைச் செய்யவள்
தவ்வையைக் காட்டி விடும்		( குறள் - 167)

```

@snap[span-100 text-08 text-left]

> பிறர் உயர்வு கண்டு பொறாமைப்படுபவனைப் பார்க்கும் திருமகள் வெறுப்புக் கொண்டு மூதேவிக்கு அவனை அடையாளம் காட்டிவிட்டு விலகிப் போய்விடுவாள்.

@snapend

### பொறாமை குணம் அவ்வளவு மோசமானதா?

---?include=includes/Alukkaramai.md

```
அழுக்காறு எனஒரு பாவி திருச்செற்றுத்
தீயுழி உய்த்து விடும்		( குறள் - 168)

```

@snap[span-100 text-08 text-left]

> பொறாமை எனப்படும் ஒப்பில்லாத பாவி எவனிடம் இருக்கிறதோ, அவனது செல்வத்தை அழிப்பதோடு, அவனை நரகத்திலும் அது சேர்க்கும்.

@snapend

---?include=includes/Alukkaramai.md

### பொறாமைப் படுபவருள் சிலர் நன்றாக வாழ்கிறார்களே ?

---?include=includes/Alukkaramai.md

```
அவ்விய நெஞ்சத்தான் ஆக்கமும் செவ்வியான்
கேடும் நினைக்கப் படும்		( குறள் - 169)

```

@snap[span-100 text-08 text-left]

> பொறாமை கொண்டவரின் உயர்வும் , அது இல்லாத நல்லவரின் தாழ்வும் ஆராயப் பட  வேண்டியது.

@snapend

### பொறாமை குணம் இருந்தால் என்ன?

---?include=includes/Alukkaramai.md

```
அழுக்கற்று அகன்றாரும் இல்லை அஃது இல்லார்
பெருக்கத்தில் தீர்ந்தாரும் இல்		( குறள் - 170)

```

@snap[span-100 text-08 text-left]

> பொறாமை கொண்டு உயர்ந்தவரும் இல்லை. அது இல்லாத போது தாழ்ந்தவரும் இல்லை.

@snapend

@snap[span-100 text-08 text-left]
<div class="conclusion" >
பொறாமை குணத்தை நீக்க என்ன செய்வது?

நம் வாழ்வை அடுத்தவர் வாழ்வோடு ஒப்பிட்டுபார்ப்பதை தவிர்க்கலாம். 

நமக்கு யார்மீது பொறாமை வருகிறதோ அவர்களை பாராட்டப் பழகலாம். 

அடுத்தவர் மீது பொறாமை பட்டு குறை சொல்லி நேரம் செலவு செய்வதை  விட்டு நமது வாழ்வின் முன்னேற்றத்திற்கு உழைக்கலாம். 

வாழ்வில் உயர்த்தவரை பார்த்து பொறாமை படாமல் அவர்களை அங்கீகரிக்கலாம். 

</div>

@snapend
